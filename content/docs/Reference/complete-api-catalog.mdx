# Complete API catalog

This page maps **every public capability** in `apexify.js` (as shipped in the installed package types) to the documentation topic that covers it. Use it when you know what you want to do but not which page to open, or when auditing whether a feature is documented.

<Alert type="info" title="Source of truth">
Method names and signatures match **`ApexPainter`** in `apexify.js` **v5.3.x** (`dist/esm/Canvas/ApexPainter.d.ts`). **`CanvasUtils`** lists follow `dist/esm/Canvas/utils/canvasUtils.d.ts`. If the package upgrades, regenerate this mapping from those files.
</Alert>

## Package exports

```typescript
import { ApexPainter, CanvasUtils, CanvasTypes } from 'apexify.js';
// Named helpers and types are also re-exported from canvasUtils (url, base64, …)
```

| Export | Role |
|--------|------|
| **`ApexPainter`** | Main class — canvas, text, images, charts, GIF, video, pixels, paths, hit tests, batch utilities. |
| **`CanvasUtils`** | Low-level helpers used by the engine and available for advanced pipelines (background drawing, shapes, filters, stitching, …). |
| **`CanvasTypes`** | Shared TypeScript types (`CanvasConfig`, `ImageProperties`, `TextProperties`, …). |

### `createVideo` reminder

`createVideo` accepts **`source`** plus **one** operation key (`getInfo`, `extractFrame`, `createFromFrames`, `convert`, …). **Encode from frames** is documented in depth under [Video processing → Create video from frames](/docs#video-processing) (search the on-page outline).

## CanvasUtils (named exports)

Use these when building custom pipelines or tooling; ApexPainter calls them internally.

**Conversions:** `url`, `arrayBuffer`, `base64`, `dataURL`, `blob`  

**Backgrounds:** `drawBackgroundColor`, `drawBackgroundGradient`, `drawBackgroundLayers`, `customBackground`, `drawPattern`, `applyNoise`, `applyCanvasZoom`, `resolveMediaPath`  

**Image drawing:** `buildPath`, `applyRotation`, `createGradientFill`, `fitInto`, `loadImageCached`, `applyStroke`, `applyShadow`, `drawBoxBackground`, `drawShape`, `isShapeSource`, `createShapePath`, `applyImageFilters`, `applyProfessionalImageFilters`  

**Masking / warp:** `applyImageMask`, `applyClipPath`, `applyPerspectiveDistortion`, `applyBulgeDistortion`, `applyMeshWarp`  

**Effects:** `applyVignette`, `applyLensFlare`, `applyChromaticAberration`, `applyFilmGrain`  

**Text (low-level):** `drawText`, `WrappedText`  

**General image:** `loadImages`, `resizingImg`, `converter`, `applyColorFilters`, `imgEffects`, `cropInner`, `cropOuter`, `detectColors`, `removeColor`, `bgRemoval`  

**Custom lines:** `customLines`, `drawArrow`, `drawMarker`, `createSmoothPath`, `createCatmullRomPath`, `applyLinePattern`, `applyLineTexture`, `getPointOnLinePath`  

**Batch / layout:** `batchOperations`, `chainOperations`, `stitchImages`, `createCollage`, `compressImage`, `extractPalette`  

**Charts namespace:** `Charts` — sub-exports for chart internals (advanced use). See [Bar charts](/docs#bar-charts) and sibling chart pages.

**Errors / canvas:** `getErrorMessage`, `getCanvasContext`

## Minimal end-to-end usage (correct imports)

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter(); // default PNG buffer output

async function demo() {
  const { buffer } = await painter.createCanvas({
    width: 400,
    height: 200,
    colorBg: '#1e293b',
  });

  const withLabel = await painter.createText(
    {
      text: 'Apexify.js',
      x: 40,
      y: 110,
      fontSize: 28,
      color: '#e2e8f0',
    },
    buffer
  );

  fs.writeFileSync('demo.png', withLabel);
}

demo();
```
</ts>
