# Charts: complete guide (every `createChart` type)

**Implementation:** `lib/Canvas/extended/ChartCreator.ts` dispatches to `lib/Canvas/utils/Charts/*.ts`. **`ApexPainter.createChart(type, data, options)`** and **`Charts.create*Chart`** return the same **PNG `Buffer`**.

Use this page when you need **one chart image**. For **two charts** on one canvas see [comparison & combo](/docs#charts-comparison-combo). For **bars + lines + dual Y** see **`createComboChart`** on that page.

**Types:** Per-chart **`data`** / **`options`** interfaces live beside each module under **`lib/Canvas/utils/Charts/`** (also re-exported from **`Charts`**). Deep contracts are in TypeScript ŌĆö treat tables below as the fields people set most often.

---

## Shared building blocks

### `dimensions`

| Field | Role |
|-------|------|
| **`width`**, **`height`** | Canvas size (defaults vary by chart; bar charts infer width from X span if omitted ŌĆö set explicitly for fixed layouts). |
| **`padding`** | **`top`**, **`right`**, **`bottom`**, **`left`** ŌĆö extra outer inset around title + legend + plot. |

### `appearance` (`ChartAppearanceExtended`)

Same vocabulary as rich canvases ŌĆö see [Charts hub](/docs#charts) **Appearance** section. Highlights:

| Field | Role |
|-------|------|
| **`backgroundColor`**, **`backgroundGradient`** | Base fill when no image. |
| **`backgroundImage`** | Full-bleed image (`resolveMediaPath`). |
| **`customBg`** | Same shape as **`CanvasConfig.customBg`** (fit, align, opacity, filters). |
| **`bgLayers`** | Stacked overlays after base. |
| **`patternBg`** | Preset / custom pattern overlay. |
| **`noiseBg`** | Grain overlay (`intensity`). |
| **`axisColor`**, **`axisWidth`**, **`arrowSize`** | Axis styling where used. |
| **`borderRadius`**, **`borderWidth`**, **`borderColor`** | Optional outer frame (line chart and others). |

### Legends (cartesian charts)

Typical keys: **`show`**, **`position`** ([legend placement](/docs#charts) ŌĆö `bottom`, `topRight`, aliases like **`top_left`**), **`entries`** (**`{ label, color?, gradient? }[]`**), **`fontSize`**, **`spacing`**, **`padding`**, **`maxWidth`**, **`wrapText`**, **`backgroundColor`**, **`textColor`**, **`textGradient`**, **`textStyle`**.

Line / scatter also support **`fit`**: **`content`** (default) vs **`stretch`**, plus **`minWidth`**, **`width`**.

### Grid

**`grid.show`**, **`grid.color`**, **`grid.width`** ŌĆö dashed/solid grid behind plot on bar / line / scatter / combo where implemented.

## Pie (`createChart('pie', ŌĆ”)`)

**Module:** `piechart.ts` ┬Ę **`PieSlice[]`** + **`PieChartOptions`**.

### Data: `PieSlice`

| Field | Role |
|-------|------|
| **`label`**, **`value`** | Required slice identity and size. |
| **`color`**, **`gradient`** | Fill (gradient wins when both set per renderer rules). |
| **`showValue`**, **`showLabel`**, **`valueLabel`** | Per-slice label overrides. |
| **`opacity`** | 0ŌĆō1 fill multiplier. |
| **`shadow`**, **`stroke`** | Drop shadow and outline (stroke supports **`gradient`**). |

### Options highlights

| Section | Role |
|---------|------|
| **`type`**, **`donutInnerRadius`** | **`'pie'`** (default) vs **`'donut'`**; inner radius ratio for donut. |
| **`labels.title`** | Chart title text + font / color / **`textStyle`**. |
| **`labels.sliceLabels`**, **`valueLabels`**, **`showValues`**, **`showLabels`**, **`valueFormat`** | Segment annotations; **`valueFormat(value, pct)`** customizes numbers. |
| **`legends.standard`** | Box legend: **`position`**, **`show?: false`** to hide, **`wrapText`**, **`maxWidth`**, colors. |
| **`legends.connected`** | Pie-only ŌĆ£connectedŌĆØ legend with leader lines (**`lineColor`**, **`lineWidth`**). |
| **`slices`** | Default **`opacity`**, **`shadow`**, **`stroke`** for all slices. |

      showValues: true,

    },

      standard: {
        position: 'bottom',
        textColor: '#e2e8f0',
        backgroundColor: 'rgba(15,23,42,0.6)',

    },

## Bar ŌĆö vertical (`createChart('bar', ŌĆ”)`)

**Module:** `barchart.ts` ┬Ę **`BarChartData[]`** + **`BarChartOptions`**.

### `BarChartType`

**`standard`** | **`grouped`** | **`stacked`** | **`waterfall`** | **`lollipop`**

Set with **`options.type`**. **`grouped`** / **`stacked`** / **`waterfall`** expect **`values: BarSegment[]`** on each row.

### Data: `BarChartData`

| Field | Role |
|-------|------|
| **`label`** | Category name (X band). |
| **`xStart`**, **`xEnd`** | Continuous X span for that bar (e.g. `0ŌĆō1`, `1ŌĆō2`). |
| **`value`** | Single column height (**`standard`**, **`lollipop`**). |
| **`values`** | **`BarSegment[]`** for grouped/stacked/waterfall (**`value`**, **`color`**, **`label`**, **`showValue`**, ŌĆ”). |
| **`color`**, **`gradient`** | Bar fill when not using per-segment colors. |
| **`labelPosition`**, **`showValue`**, **`valueColor`** | Annotation controls. |

### Options highlights

| Section | Role |
|---------|------|
| **`waterfall.initialValue`** | Starting baseline for waterfall sums. |
| **`axes.x`**, **`axes.y`** | **`label`**, **`range`**, **`values`**, **`tickFontSize`**, **`baseline`** on **Y**. |
| **`labels.barLabelDefaults`**, **`valueLabelDefaults`** | Default annotation behavior. |
| **`legend`** | Entries + placement (see shared). |
| **`bars.spacing`**, **`groupSpacing`**, **`segmentSpacing`**, **`minWidth`**, **`lineWidth`**, **`dotSize`** | Geometry (**`lollipop`** uses line + dot). |

## Line (`createChart('line', ŌĆ”)`)

**Module:** `linechart.ts` ┬Ę **`LineSeries[]`** + **`LineChartOptions`**.

### `LineSeries`

| Field | Role |
|-------|------|
| **`label`**, **`data`** | **`LineDataPoint[]`** with **`x`**, **`y`**, optional **`label`**, **`errorBar`**. |
| **`color`**, **`gradient`**, **`lineWidth`**, **`lineStyle`** | Stroke (**`solid`**, **`dashed`**, **`step`**, ŌĆ”). |
| **`smoothness`** | **`none`**, **`bezier`**, **`spline`**. |
| **`smoothnessTension`** | 0ŌĆō1 curve amount for bezier-style paths. |
| **`marker`** | **`type`**: circle/square/triangle/ŌĆ”, **`size`**, **`color`**, **`filled`**, **`show`**. |
| **`area`** | **`type`**: **`below`** / **`above`** / **`between`** / **`around`** / **`none`**; **`color`**, **`opacity`**; **`secondLine`** for bands. |
| **`correlation`** | Trend overlay: **`linear`**, **`polynomial`**, **`exponential`**, **`logarithmic`**, **`none`**. |
| **`opacity`** | Whole-series multiplier. |

### Axes (`AxisConfig`)

Supports **`range`**, **`scale: 'linear' | 'log'`**, **`dateTime`**, **`dateFormat`**, **`showTickMarks`**, **`showTickLabels`**.

## Radar (`createChart('radar', ŌĆ”)`)

**Module:** `radarchart.ts` ┬Ę **`RadarSeries[]`** + **`RadarChartOptions`**.

**Rules:**

- **`options.radar.categories`** ŌĆö **Ōēź 3** axis labels.
- Each **`RadarSeries.values`** must have **exactly `categories.length`** numbers.

### `radar` option bucket

| Field | Role |
|-------|------|
| **`maxValue`** | Scale ceiling (defaults ~ max ├Ś 1.05). |
| **`gridLevels`** | Number of concentric rings. |
| **`fill`**, **`fillOpacity`** | Polygon fill (series can override **`fillOpacity`**). |
| **`showPoints`**, **`pointRadius`** | Vertices. |
| **`gridColor`**, **`gridWidth`**, **`axisLabelFontSize`**, **`axisLabelColor`** | Cosmetics. |
| **`labelMarginRatio`** | Space outside outer labels. |
| **`opacity`** | Default series opacity; **`RadarSeries.opacity`** overrides per series. |

## Polar area (`createChart('polarArea', ŌĆ”)`)

**Module:** `polarareachart.ts` ┬Ę **`PolarAreaSlice[]`** + **`PolarAreaChartOptions`**.

Slices behave like a **polar bar** chart: each sliceŌĆÖs **angular width** is uniform; **radius** encodes **`value`**.

| Option | Role |
|--------|------|
| **`scale`** | **`radius`** (linear radius) vs **`area`** (radius ŌłØ ŌłÜvalue so areas feel proportional). |
| **`polar.innerRadiusRatio`** | Donut-style hole (0 = solid center). |
| **`polar.startAngleDeg`** | Rotation (default **-90** = first slice from top). |
| **`polar.labelBandRatio`** | Space for outer labels. |
| **`polar.sliceStrokeWidth`**, **`sliceStrokeColor`** | Outlines. |
| **`labels.showValues`**, **`valueFormat`**, **`sliceLabelFontSize`** | Value annotations on slices. |

All **`value`** entries must sum to a **positive** total.

<ts>
