# Getting started

**Apexify.js** is a Node.js canvas toolkit built on **`@napi-rs/canvas`**. You compose PNG (and related) outputs with **`ApexPainter`**, optional FFmpeg for video, and Sharp-backed helpers for raster work.

## Install

<CodeBlock lang="bash">
```bash
npm install apexify.js
```
</CodeBlock>

- **Node.js** 16+
- **TypeScript** recommended (`CanvasTypes` mirrors contracts)

## Minimal usage

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter(); // default output channel for `outPut()` — see [Package surface](/docs#package-surface)

const { buffer } = await painter.createCanvas({
  width: 640,
  height: 360,
  colorBg: '#0f172a',
});

const withTitle = await painter.createText(
  [{ text: 'Hello', x: 48, y: 200, font: { size: 42 }, color: '#e2e8f0' }],
  buffer,
);

fs.writeFileSync('out.png', withTitle);
```
</ts>
<js>
```javascript
const { ApexPainter } = require('apexify.js');
const fs = require('fs');

(async () => {
  const painter = new ApexPainter();
  const { buffer } = await painter.createCanvas({
    width: 640,
    height: 360,
    colorBg: '#0f172a',
  });
  const withTitle = await painter.createText(
    [{ text: 'Hello', x: 48, y: 200, font: { size: 42 }, color: '#e2e8f0' }],
    buffer,
  );
  fs.writeFileSync('out.png', withTitle);
})();
```
</js>
</CodeSwitcher>

## npm entry note

`apexify.js` **`lib/index.ts`** may call **`fetch`** once against the npm registry to log if a newer version exists — expect a harmless network attempt in online CI; offline builds can ignore failed checks.
