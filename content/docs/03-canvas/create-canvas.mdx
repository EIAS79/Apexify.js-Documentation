# Create canvas (`createCanvas`)

**API:** `await painter.createCanvas(canvas: CanvasConfig)` → **`CanvasResults`**.

**Implementation:** `lib/Canvas/extended/CanvasCreator.ts` draws in a fixed order using **`lib/Canvas/utils/Background/bg.ts`** (solid / gradient / image / layers / bitmap pattern / noise / zoom) plus **`lib/Canvas/utils/Patterns/enhancedPatternRenderer.ts`** for **`patternBg`** and **`presetPattern`** layers.

<Alert type="info" title="Deep references">
Draw order and validation: [Canvas pipeline](/docs#canvas-pipeline). Background primitives: **`lib/Canvas/utils/Background/bg.ts`**. Procedural meshes: [Canvas patterns, noise & transforms](/docs#canvas-pattern-noise-transforms). Types: **`CanvasConfig`**, **`BackgroundLayer`**, **`PatternOptions`** in **`CanvasTypes`** ([Canvas utils & types](/docs#canvas-utils-and-types)).
