## End-to-end composition example

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const { buffer } = await painter.createCanvas({
  width: 900,
  height: 500,
  gradientBg: {
    type: 'radial',
    colors: [
      { stop: 0, color: '#1e1b4b' },
      { stop: 1, color: '#312e81' },
    ],
  },
  bgLayers: [
    { type: 'noise', intensity: 0.04, blendMode: 'overlay' },
  ],
  patternBg: {
    type: 'dots',
    color: '#c7d2fe',
    opacity: 0.15,
    size: 8,
    spacing: 28,
  },
  borderRadius: 24,
  stroke: { color: '#6366f1', width: 3 },
});

fs.writeFileSync('hero-bg.png', buffer);
```
</ts>
<js>
```javascript
const { ApexPainter } = require('apexify.js');
const fs = require('fs');

(async () => {
  const painter = new ApexPainter();
  const { buffer } = await painter.createCanvas({
    width: 900,
    height: 500,
    gradientBg: {
      type: 'radial',
      colors: [
        { stop: 0, color: '#1e1b4b' },
        { stop: 1, color: '#312e81' },
      ],
    },
    bgLayers: [{ type: 'noise', intensity: 0.04, blendMode: 'overlay' }],
    patternBg: {
      type: 'dots',
      color: '#c7d2fe',
      opacity: 0.15,
      size: 8,
      spacing: 28,
    },
    borderRadius: 24,
    stroke: { color: '#6366f1', width: 3 },
  });
  fs.writeFileSync('hero-bg.png', buffer);
})();
```
</js>
</CodeSwitcher>

## Using `CanvasUtils` directly (advanced)

For custom **`SKRSContext2D`** pipelines you can call **`drawBackgroundColor`**, **`drawBackgroundGradient`**, **`customBackground`**, **`drawBackgroundLayers`**, **`drawPattern`**, **`applyNoise`**, **`applyCanvasZoom`** from **`apexify.js`** — same primitives **`CanvasCreator`** uses (**`lib/Canvas/utils/Background/bg.ts`**).
