# Horizontal Bar Charts

Apexify.js provides comprehensive horizontal bar chart generation with support for multiple chart types (standard, grouped, stacked, lollipop), gradients, enhanced text styling, and customizable axes.

## Overview

Horizontal bar charts are created using the unified `createChart()` method with `chartType: 'horizontalBar'`. They support 4 types: standard, grouped, stacked, and lollipop. Charts are fully responsive with automatic height calculation.

## Chart Types

- **Standard**: Single value per category
- **Grouped**: Multiple series grouped side-by-side
- **Stacked**: Multiple series stacked on top of each other
- **Lollipop**: Bars with circular markers at the end

## Basic Usage

### Standard Horizontal Bar Chart

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const data = [
  {
    label: 'Product A',
    value: 150,
    color: '#4A90E2'
  },
  {
    label: 'Product B',
    value: 200,
    color: '#50C878'
  },
  {
    label: 'Product C',
    value: 120,
    color: '#FF6B6B'

```

## Data Structure

### Standard Horizontal Bar Chart

<CodeSwitcher>
<ts>
```typescript
interface HorizontalBarChartData {
  label: string;              // Category label (displayed on Y-axis)
  value?: number;             // Bar value (for standard charts)
  values?: HorizontalBarSegment[]; // Array of segments (for grouped/stacked)
  xStart?: number;            // Optional: X-axis start position
  xEnd?: number;              // Optional: X-axis end position
  color?: string;             // Bar color (overridden by gradient if provided)
  gradient?: gradient;        // Gradient fill (overrides color)
  labelColor?: string;        // Label text color
  labelPosition?: 'top' | 'left' | 'right' | 'inside' | 'bottom';
  valueColor?: string;        // Value label text color
  showValue?: boolean;        // Show/hide value label
}
```

  labelPosition: 'left',      // Optional: Label position
  valueColor: '#000',         // Optional: Value label color
