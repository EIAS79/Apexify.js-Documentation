# Line Charts

Apexify.js provides powerful line chart generation with support for multiple datasets, smooth curves, filled areas, plot points, and comprehensive styling options.

## Overview

Line charts are created using `createChart()` with `chartType: 'line'` and `chartNumber: 1`. Support multiple lines, smooth curves, filled areas, and customizable grids.

## Basic Line Chart

Create a simple line chart:

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const chart = await painter.createChart({
  data: [
    [
      { label: 'Jan', y: 30 },
      { label: 'Feb', y: 45 },
      { label: 'Mar', y: 35 },
      { label: 'Apr', y: 60 },
      { label: 'May', y: 50 }
    ]
  ],
  lineConfig: {
    yLabels: ['0', '20', '40', '60', '80', '100'],
    fillArea: [{ color: 'rgba(255, 107, 107, 0.2)' }],
    lineColor: ['#ff6b6b'],
    lineTension: [0.3],
    plot: {
      enable: true,
      color: ['#ff6b6b'],
      size: 5
    },
    grid: {
      type: 'both',
      color: '#e0e0e0',
      width: 1
    },
    canvas: {
      width: 800,
      height: 600,
      bgColor: '#ffffff'
    }
  }
}, {
  chartType: 'line',
  chartNumber: 1
});

fs.writeFileSync('line-chart.png', chart);
```
</ts>

## Multiple Lines

Create charts with multiple datasets:

<CodeSwitcher>
<ts>
```typescript
const chart = await painter.createChart({
  data: [
    // First line
    [
      { label: 'Jan', y: 30 },
      { label: 'Feb', y: 45 },
      { label: 'Mar', y: 35 }
    ],
    // Second line
    [
      { label: 'Jan', y: 20 },
      { label: 'Feb', y: 35 },
      { label: 'Mar', y: 50 }
    ],
    // Third line
    [
      { label: 'Jan', y: 40 },
      { label: 'Feb', y: 30 },
      { label: 'Mar', y: 45 }
    ]
  ],
  lineConfig: {
    yLabels: ['0', '20', '40', '60', '80', '100'],
    fillArea: [
      { color: 'rgba(255, 107, 107, 0.2)' },
      { color: 'rgba(78, 205, 196, 0.2)' },
      { color: 'rgba(69, 183, 209, 0.2)' }
    ],
    lineColor: ['#ff6b6b', '#4ecdc4', '#45b7d1'],
    lineTension: [0.3, 0.3, 0.3],
    plot: {
      enable: true,
      color: ['#ff6b6b', '#4ecdc4', '#45b7d1'],
      size: 5
    },
    grid: {
      type: 'both',
      color: '#e0e0e0',
      width: 1
    },
    canvas: {
      width: 1200,
      height: 800,
      bgColor: '#ffffff'
    }
  }
}, {
  chartType: 'line',
  chartNumber: 1
});
```

## Line Configuration

### Grid Types

<CodeSwitcher>
<ts>
```typescript
grid: {
  type: 'vertical' | 'horizontal' | 'both',  // Grid type
  color: '#e0e0e0',                          // Grid line color
  width: 1                                   // Grid line width
}
```
</ts>
<js>
```javascript
grid: {
```

</CodeSwitcher>

### Line Tension

<CodeSwitcher>
<ts>
```typescript

```

<js>
```javascript
lineTension: [0.1, 0.3, 0.5]  // One value per line
```

## Complete Example

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const chart = await painter.createChart({
  data: [
    [
      { label: 'Q1', y: 45 },
      { label: 'Q2', y: 62 },
      { label: 'Q3', y: 38 },
      { label: 'Q4', y: 71 }
    ],
    [
      { label: 'Q1', y: 30 },
      { label: 'Q2', y: 55 },
      { label: 'Q3', y: 48 },
      { label: 'Q4', y: 65 }
    ]
  ],
  lineConfig: {
    yLabels: ['0', '20', '40', '60', '80', '100'],
    fillArea: [
      { color: 'rgba(255, 107, 107, 0.15)' },
      { color: 'rgba(78, 205, 196, 0.15)' }
    ],
    lineColor: ['#ff6b6b', '#4ecdc4'],
    lineTension: [0.4, 0.4],
    plot: {
      enable: true,
      color: ['#ff6b6b', '#4ecdc4'],
      size: 6
    },
    grid: {
      type: 'both',
      color: '#dee2e6',
      width: 1
    },
    yaxisLabel: {
      label: 'Revenue (in thousands)',
      color: '#495057',
      fontSize: '18px'
    },
    keys: {
      '#ff6b6b': '2023 Sales',
      '#4ecdc4': '2024 Sales'
    },
    keysConfig: {
      radius: 12,
      keyPadding: 40,
      textPadding: 70,
      lineWidth: 4,
      fontColor: '#2c3e50'
    },
    canvas: {
      width: 1200,
      height: 800,
      bgColor: '#f8f9fa',
      fontColor: '#495057',
      fontSize: 16
    }
  }
}, {
  chartType: 'line',
  chartNumber: 1
});

fs.writeFileSync('revenue-comparison.png', chart);
```
</ts>
