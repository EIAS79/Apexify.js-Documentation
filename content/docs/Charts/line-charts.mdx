# Line Charts

Apexify.js provides powerful line chart generation with support for multiple series, smooth curves, markers, filled areas, error bars, correlation lines, and comprehensive styling options.

## Overview

Line charts are created using the unified `createChart()` method with `chartType: 'line'`. Support multiple lines/series, smooth curves (bezier, spline), filled areas, error bars, correlation/regression lines, and customizable markers.

## Basic Usage

## Data Structure

### Line Series

<CodeSwitcher>
<ts>
```typescript
interface LineDataPoint {
  x: number;                  // X-axis value
  y: number;                  // Y-axis value
  label?: string;             // Optional label for this point
  markerColor?: string;       // Optional custom marker color
  showMarker?: boolean;       // Override marker visibility
  errorBar?: ErrorBarConfig;  // Error bar for this point
}

interface LineSeries {
  label: string;              // Series label (for legend)
  data: LineDataPoint[];      // Array of data points
  color?: string;             // Line color (overridden by gradient if provided)
  gradient?: gradient;        // Gradient for line (overrides color)
  lineWidth?: number;         // Line width (default: 2)
  lineStyle?: LineStyle;      // 'solid' | 'dashed' | 'dotted' | etc.
  smoothness?: SmoothnessType; // 'none' | 'bezier' | 'spline'
  showLine?: boolean;         // Show/hide line (default: true)
  marker?: {
    type?: MarkerType;        // 'circle' | 'square' | 'triangle' | 'diamond' | 'cross' | 'none'
    size?: number;            // Marker size (default: 6)
    color?: string;           // Marker color
    show?: boolean;           // Show/hide markers (default: true)
    filled?: boolean;         // Fill markers (default: true)
  };
  errorBar?: {
    show?: boolean;           // Show error bars (default: false)
    color?: string;           // Error bar color
    width?: number;           // Error bar line width
    capSize?: number;         // Error bar cap size
  };
  area?: AreaConfig;          // Area shading configuration
  correlation?: {
    type?: CorrelationType;   // 'none' | 'linear' | 'polynomial' | 'exponential' | 'logarithmic'
    degree?: number;          // Polynomial degree (for polynomial regression)
    color?: string;           // Correlation line color
    lineWidth?: number;       // Correlation line width
    lineStyle?: LineStyle;    // Correlation line style
    show?: boolean;           // Show/hide correlation line
  };
}
```

### Area Types

- **below**: Fill area below the line (to baseline/x-axis)
- **above**: Fill area above the line (to baseline/x-axis)
- **between**: Fill area between two lines (requires `secondLine`)
- **around**: Fill area around confidence interval (requires `upperBound` and `lowerBound`)
- **none**: No area filling
