# Pie Charts

Apexify.js provides comprehensive pie chart generation with customizable slices, labels, legends, gradients, and enhanced text styling. Create professional pie charts (including donut charts) to visualize proportional data.

## Overview

Pie charts are created using the unified `createChart()` method with `chartType: 'pie'`. Each slice represents a portion of the whole, with automatic percentage calculation. Pie charts support gradients, custom fonts, enhanced text styling, and two types of legends.

## Basic Usage

Create a simple pie chart:

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const chart = await painter.createChart('pie', [
  { label: 'Desktop', value: 45, color: '#ff6b6b' },
  { label: 'Mobile', value: 30, color: '#4ecdc4' },
  { label: 'Tablet', value: 15, color: '#45b7d1' },
  { label: 'Other', value: 10, color: '#ffd700' }
]);

fs.writeFileSync('pie-chart.png', chart);
```
</ts>
<js>
```javascript

    { label: 'Other', value: 10, color: '#ffd700' }
  ]);
```

## Data Structure

Each pie slice is defined as a `PieSlice` object:

<CodeSwitcher>
<ts>
```typescript
interface PieSlice {
  label: string;              // Display label for the slice
  value: number;              // Numeric value (percentage calculated automatically)
  color?: string;             // Solid color (overridden by gradient if provided)
  gradient?: gradient;        // Gradient fill for slice (overrides color)
  showValue?: boolean;        // Show/hide value label on slice (default: true)
  showLabel?: boolean;        // Show/hide slice label on slice (default: false)
  valueLabel?: string;        // Custom value label (default: auto-generated)
  opacity?: number;           // Slice opacity (0-1, default: 1)
  shadow?: {
    color?: string;
    offsetX?: number;
    offsetY?: number;
    blur?: number;
  };
  stroke?: {
    color?: string;
    width?: number;
    gradient?: gradient;      // Stroke gradient (overrides color)
  };
}
```
</ts>
<js>
```javascript
// Pie slice object
{
  label: 'Category Name',     // Display label for the slice
  value: 45,                  // Numeric value (percentage calculated automatically)
  color: '#ff6b6b',           // Optional: Solid color
  gradient: { ... },          // Optional: Gradient fill (overrides color)
  showValue: true,            // Optional: Show value label (default: true)
  showLabel: false,           // Optional: Show label on slice (default: false)
  opacity: 1,                 // Optional: Slice opacity 0-1 (default: 1)
  shadow: { ... },            // Optional: Shadow configuration
  stroke: { ... }             // Optional: Stroke configuration
}
```
</js>

### Donut Charts

Donut charts are a variation of pie charts with a hollow center. They're created by setting `type: 'donut'` in the options. All pie chart features (gradients, labels, legends, enhanced text styling) work with donut charts.

<CodeSwitcher>
<ts>
```typescript
// Basic donut chart

  type: 'donut',
  donutInnerRadius: 0.6       // Inner radius ratio 0-1 (default: 0.6)
});

// Advanced donut chart with custom styling
const advancedDonut = await painter.createChart('pie', data, {
  type: 'donut',
  donutInnerRadius: 0.5,      // Larger hole (50% of radius)
  dimensions: {
    width: 1000,
    height: 800
```

**Donut Inner Radius:**
- `donutInnerRadius` controls the size of the hollow center
- Value range: `0.0` to `1.0` (ratio of inner radius to outer radius)
- `0.6` = 60% of the radius (default, creates a moderate donut hole)
- `0.3` = 30% (thin ring, larger hole)
- `0.8` = 80% (thick ring, smaller hole)
