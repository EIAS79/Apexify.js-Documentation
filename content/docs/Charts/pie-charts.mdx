# Pie Charts

Apexify.js provides comprehensive pie chart generation with customizable slices, labels, legends, and styling. Create professional pie charts to visualize proportional data.

## Overview

Pie charts are created using `createChart()` with `chartType: 'pie'` and `chartNumber: 1`. Each slice represents a portion of the whole, with automatic percentage calculation.

## Basic Pie Chart

Create a simple pie chart:

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const chart = await painter.createChart({
  data: [
    { label: 'Desktop', color: '#ff6b6b', value: 45, key: 'Desktop' },
    { label: 'Mobile', color: '#4ecdc4', value: 30, key: 'Mobile' },
    { label: 'Tablet', color: '#45b7d1', value: 15, key: 'Tablet' },
    { label: 'Other', color: '#ffd700', value: 10, key: 'Other' }
  ],
  pieConfig: {
    canvas: {
      width: 1200,
      height: 600,
      bgcolor: '#ffffff'
    }
  }
}, {
  chartType: 'pie',
  chartNumber: 1
});

fs.writeFileSync('pie-chart.png', chart);
```
</ts>
<js>

### Pie Data Configuration

<CodeSwitcher>
<ts>
```typescript
pieConfig: {
  pieData: {
    x: 0,              // X offset from center
    y: 0,              // Y offset from center
    radius: 200,       // Pie radius (default: auto-calculated)
    stroke: {
      color: '#000000', // Slice border color
      size: 2           // Slice border width
    },
    title: {
      text: 'Chart Title',
      color: '#000000',
      fontSize: 24,
      x: 0,            // Title X offset
      y: 0             // Title Y offset
    },
    boxes: {
      labelDistance: 50,    // Distance from pie edge to label box
      width: 100,           // Label box width
      height: 40,           // Label box height
      fontSize: 14,         // Label font size
      labelColor: '#ffffff', // Label text color
      boxColor: '#000000',   // Label box background
      strokeColor: '#000000' // Label line color
    }
  }
}
```
</ts>

<Alert type="info" title="Automatic Percentage Calculation">
The pie chart automatically calculates slice sizes based on the ratio of each value to the total. You don't need to provide percentages - just provide the raw values.
