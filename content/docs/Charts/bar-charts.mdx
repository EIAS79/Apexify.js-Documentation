# Bar Charts

Apexify.js provides powerful bar chart generation with support for multiple chart types (standard, grouped, stacked, waterfall, lollipop), gradients, enhanced text styling, customizable axes, legends, and grids.

## Overview

Bar charts are created using the unified `createChart()` method with `chartType: 'bar'`. Bar charts support 5 types: standard, grouped, stacked, waterfall, and lollipop. Charts are fully responsive with automatic width calculation.

## Chart Types

- **Standard**: Single value per category
- **Grouped**: Multiple series grouped side-by-side
- **Stacked**: Multiple series stacked on top of each other
- **Waterfall**: Shows cumulative effect of values
- **Lollipop**: Bars with circular markers at the end

import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const data = [
  {
    label: 'Jan',
    value: 45,
    xStart: 0,
    xEnd: 1,
    color: '#4A90E2'
  },
  {
    label: 'Feb',

    xEnd: 2,
    color: '#50C878'
  },
  {
    label: 'Mar',
    value: 55,
    xStart: 2,
