# Bar Charts

Apexify.js provides powerful bar chart generation capabilities through the `createChart()` method. Create professional vertical bar charts with customizable styling, backgrounds, grids, and labels.

## Overview

Bar charts are created using `createChart()` with `chartType: 'bar'` and `chartNumber: 1`. The method accepts comprehensive configuration for data, styling, and layout.

## Basic Bar Chart

Create a simple bar chart with minimal configuration:

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const chart = await painter.createChart({
  chartData: {
    width: 800,
    height: 600,
    title: {
      title: 'Sales Data',
      color: '#000000',
      size: 24
    },
    bg: {
      bgColor: '#ffffff'
    }
  },
  xLabels: [0, 100, 200, 300, 400, 500],
  yLabels: [0, 20, 40, 60, 80, 100],
  data: {
    xAxis: [
      {
        label: 'Q1',
        value: 45,
        position: { startsXLabel: 0, endsXLabel: 100 }
      },
      {
        label: 'Q2',
        value: 65,
        position: { startsXLabel: 100, endsXLabel: 200 }
      },
      {
        label: 'Q3',
        value: 35,
        position: { startsXLabel: 200, endsXLabel: 300 }
      },
      {
        label: 'Q4',
        value: 85,
        position: { startsXLabel: 300, endsXLabel: 400 }
      }
    ],
    yAxis: [0, 20, 40, 60, 80, 100],
    xTitle: 'Quarters',
    yTitle: 'Sales (in thousands)'
  }
}, {
  chartType: 'bar',
  chartNumber: 1
});

fs.writeFileSync('bar-chart.png', chart);
```
</ts>

## Chart Configuration

### Chart Data

The `chartData` object controls the overall chart appearance:

<CodeSwitcher>
<ts>
```typescript
const chartData = {
  width: 1200,        // Canvas width
  height: 800,         // Canvas height
  widthPerc: 0.8,      // Chart width percentage (0-1)
  heightPerc: 0.8,     // Chart height percentage (0-1)
  title: {
    title: 'My Chart',
    color: '#000000',
    size: 28
  },
  bg: {
    bgColor: '#ffffff',  // Background color
    image: './bg.jpg'     // Or use background image
  },
  grid: {
    enable: true,
    color: '#e0e0e0',
    width: 1
  },
  axis: {
    color: '#000000',
    size: 2
  },
  labels: {
    color: '#666666',
    fontSize: 12
  }
};
```
</ts>

### Bar Data

Each bar is defined with position, value, and styling:

<CodeSwitcher>
<ts>
```typescript
const barData = {
  label: 'Q1',                    // Bar label
  value: 45,                      // Bar value (Y-axis)
  barColor: '#ff6b6b',            // Bar fill color
  position: {
    startsXLabel: 0,              // Start X position
    endsXLabel: 100               // End X position
  },
  stroke: {
    color: '#000000',             // Bar border color
    width: 2                      // Bar border width
  }
};
```

<Alert type="info" title="X and Y Labels">
The `xLabels` and `yLabels` arrays define the scale of your chart. The `position.startsXLabel` and `position.endsXLabel` values must fall within the range of your `xLabels` array, and `value` must fall within the range of your `yLabels` array.
</Alert>
