# Comparison Charts

Apexify.js provides powerful comparison chart functionality that allows you to compare any two chart types side by side or top-to-bottom. Each chart can be of any type (pie, bar, horizontalBar, line, donut) with its own data, configuration, and legend.

## Overview

Comparison charts are created using the `createComparisonChart()` method. This feature enables you to create side-by-side or top-bottom layouts comparing different datasets or chart types, perfect for data analysis and presentation.

## Supported Chart Combinations

You can compare any combination of chart types:
- Pie vs Bar
- Line vs Bar
- Pie vs Line
- Line vs Line
- Bar vs Bar
- Pie vs Pie
- Pie vs Donut
- Horizontal Bar vs Bar
- Horizontal Bar vs Horizontal Bar
- Horizontal Bar vs Line
- Horizontal Bar vs Pie

## Basic Usage

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

// Pie chart data
const pieData = [
  { label: 'A', value: 30, color: '#ff6b6b' },
  { label: 'B', value: 40, color: '#4ecdc4' },
  { label: 'C', value: 30, color: '#45b7d1' }
];

// Bar chart data
const barData = [
  { label: 'Jan', value: 45, xStart: 0, xEnd: 1, color: '#4A90E2' },
  { label: 'Feb', value: 65, xStart: 1, xEnd: 2, color: '#50C878' },
  { label: 'Mar', value: 55, xStart: 2, xEnd: 3, color: '#FF6B6B' }
];

const chart = await painter.createComparisonChart({
  layout: 'sideBySide',        // 'sideBySide' | 'topBottom'
  spacing: 20,                 // Spacing between charts
  chart1: {
    type: 'pie',
    data: pieData,
    options: {
      dimensions: { width: 400, height: 400 }
    },
    title: {
      text: 'Quarter 1',
      fontSize: 20,
      color: '#333333'
    }
  },
  chart2: {
    type: 'bar',
    data: barData,
    options: {
      dimensions: { height: 400 }
    },
    title: {
      text: 'Quarter 2',
      fontSize: 20,
      color: '#333333'
    }
  }
});

fs.writeFileSync('comparison-chart.png', chart);
```
</ts>

  const chart = await painter.createComparisonChart({
    layout: 'sideBySide',
    spacing: 20,
    chart1: {
      type: 'pie',
      data: pieData,
      options: {
        dimensions: { width: 400, height: 400 }

        text: 'Quarter 2',
        fontSize: 20,
        color: '#333333'
      }
    }
  });

  fs.writeFileSync('comparison-chart.png', chart);
}

createComparison();

## Individual Chart Configuration

Each chart in the comparison has its own configuration:

<CodeSwitcher>
<ts>
```typescript
const chart = await painter.createComparisonChart({
  chart1: {
    type: 'pie',               // Chart type
    data: pieData,             // Chart data
    options: {                 // Chart-specific options (same as createChart options)
      dimensions: { width: 400, height: 400 },
      labels: {
        title: {
          text: 'Chart 1 Title',
          fontSize: 20
        }
      }
      // ... all chart options available
    },
    title: {                   // Individual chart title (optional, overrides options.title)
      text: 'Q1 2024',
      fontSize: 18,
      color: '#333333',
      gradient: { ... },       // Optional
      textStyle: { ... }       // Optional
    },
    barType: 'standard',       // For bar charts: 'standard' | 'grouped' | 'stacked' | etc.
    lineStyle: 'solid',        // For line charts: 'solid' | 'dashed' | etc.
    lineSmoothness: 'bezier'   // For line charts: 'none' | 'bezier' | 'spline'
  },
  chart2: {
    type: 'bar',
    data: barData,
    options: { ... },
    title: { ... },
    barType: 'grouped'         // Bar chart type specification
  }
});
```

## Features

- **Any Chart Type Combination**: Compare pie, bar, horizontalBar, line, or donut charts
- **Flexible Layouts**: Side-by-side or top-bottom arrangements
- **Individual Configuration**: Each chart maintains its own data, options, and legend
- **Shared Background**: Common background styling across both charts
- **General Title**: Overall title for the comparison
- **Full Feature Support**: All chart features available (gradients, enhanced text styling, etc.)
- **Customizable Spacing**: Control spacing between charts
