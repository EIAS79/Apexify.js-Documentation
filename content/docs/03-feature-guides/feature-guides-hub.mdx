## Conventions

- Prefer **several small pages** over one long page.
- Link **back** to the overview hub at the top of each page.
- **`{#anchors}`** on hubs/chapters match each page **`basename`** (for example **`02-save-disk.mdx`** → **`/docs#02-save-disk`** under **`raster-batch-output/`**).
