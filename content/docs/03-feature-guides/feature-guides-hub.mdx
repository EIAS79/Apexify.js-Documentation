## Conventions

- Prefer **several small pages** over one long page.
- Link **back** to the overview hub at the top of each page.
- **`{#anchors}`** on hubs/chapters match each page **`basename`** (for example **`01-save-disk-and-multiple.mdx`** → **`/docs#01-save-disk-and-multiple`**).
