# Hit testing — paths & regions {#02-hit-testing-paths}

Back: [Raster overview](/docs#00-advanced-raster-overview) · [Path2D · `drawPath`](/docs#04-path2d-draw-path)

**`ApexPainter`** wraps **`HitDetectionCreator`** (**`lib/Canvas/extended/HitDetectionCreator.ts`**). Methods are **`async`** on the painter but the underlying logic is **synchronous**; the **`Promise`** resolves immediately.

---

## `HitDetectionResult`

| Field | Meaning |
|-------|---------|
| **`hit`** | **`true`** if the point matches the region / path test |
| **`hitType`** | **`'fill'`**, **`'stroke'`**, or **`'outside'`** (when **`hit`** is false, expect **`'outside'`**) |
| **`distance`** | Populated for some region types when **`hit`** is true (e.g. circle stores centre distance); often **`0`** or **`undefined`** when not computed |
| **`hitRegion`** | Set by **`isPointInAnyRegion`** — index into the **`regions`** array |

---

## `HitDetectionOptions`

| Option | Role |
|--------|------|
| **`includeStroke`** | When **`true`** together with **`strokeWidth`**, attempts stroke-vs-fill discrimination **where implemented** |
| **`strokeWidth`** | Numeric stroke thickness used by **rect** (stroke band) and **circle** (annulus) logic; also required for **`isPointInPath`** stroke branch |
| **`tolerance`** | Extra slack on **rect** stroke detection (default **`0`**) |
| **`fillRule`** | Passed through on **`path`** **`HitRegion`** into **`isPointInPath`** (**`'nonzero'`** default) |

The **`region`** field on **`HitDetectionOptions`** is **not** read by **`isPointInPath`** itself — attach geometry via **`HitRegion`** APIs below.

---

## `isPointInPath(path, x, y, options?)`

- **`path`**: **`Path2D`** instance **or** **`PathCommand[]`** (see **`buildPath2DFromCommands`** in **`pathCmd.ts`**).
- Uses a small off-screen canvas **`isPointInPath`** / **`isPointInStroke`** (**`@napi-rs/canvas`**).

**Stroke caveat:** The implementation gates stroke testing on **`includeStroke`** and a truthy **`strokeWidth`**, but **`ctx.lineWidth`** is **not** set from **`strokeWidth`** before **`isPointInStroke`**. Expect stroke hits to follow the context **default** line width unless you rely primarily on **fill** hits or approximate strokes with thicker **`drawPath`** + separate testing.

**`distance`:** **`0`** when **`hit`**, otherwise **`undefined`** in current code.

---

## `HitRegion` union

| **`type`** | Behaviour |
|------------|-----------|
| **`rect`** | Axis-aligned rectangle **`x,y,width,height`**. With **`includeStroke`** + **`strokeWidth`**, **`hitType`** distinguishes stroke band vs interior |
| **`circle`** | Centre **`x,y`**, **`radius`**. Stroke ring uses **`strokeWidth`** annulus when options enabled |
| **`ellipse`** | **`radiusX`**, **`radiusY`**, optional **`rotation`** (radians). **Fill** hit-test only in **`isPointInRegion`** — **`includeStroke`** does **not** refine **`hitType`** here |
| **`polygon`** | **`points[]`** (≥ **3**), winding ray-cast. **`hitType`** stays **`fill`** when **`hit`** |
| **`path`** | Delegates to **`isPointInPath`** with **`region.path`** and **`fillRule`** |
| **`custom`** | **`check(x, y) => boolean`** — your arbitrary logic |

---

## `isPointInRegion` / `isPointInAnyRegion`

- **`isPointInRegion(region, x, y, options?)`** — single **`HitRegion`**.
- **`isPointInAnyRegion(regions, x, y, options?)`** — sequential scan; first hit wins; **`hitRegion`** is the winning index.

---

## `getDistanceToRegion(region, x, y)`

Minimum distance to the **boundary** (outside distance-to-edge helpers):

| **`type`** | Supported |
|------------|-----------|
| **`rect`** | ✅ axis-aligned distance to edge |
| **`circle`** | ✅ **`abs(distanceToCentre − radius)`** |
| **`ellipse`** | ✅ approximate boundary distance (**comment in source**: simplified) |
| **`polygon`** | ✅ minimum distance to segments |
| **`path`**, **`custom`** | Returns **`0`** (no metric) |

Useful for snap-to-outline UX **only** for **`rect` / `circle` / `ellipse` / `polygon`**.

---

```typescript
const inside = await painter.isPointInRegion(
  { type: 'rect', x: 10, y: 10, width: 100, height: 40 },
  50,
  25
);
console.log(inside.hit, inside.hitType);
```
