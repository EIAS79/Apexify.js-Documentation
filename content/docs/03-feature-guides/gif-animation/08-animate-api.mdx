# `animate` — programmatic frame sequences {#08-animate-api}

Back: [GIF overview](/docs#00-create-gif-overview)

**`await painter.animate(frames, defaultDuration, defaultWidth?, defaultHeight?, options?)`** builds **each frame on a canvas** from declarative **`Frame`** objects, then either returns an array of **PNG `Buffer`s** or streams frames into an **animated GIF file** via **`gifencoder`**.

This is **not** **`createGIF`**: **`createGIF`** consumes prepared **`GIFInputFrame`** bitmaps and supports transparency presets, disposal, watermarks, **`onStart`** streaming encoders, etc. **`animate`** is a **lighter procedural animator**: describe backgrounds, optional **`source`** images, transforms, and **`onDrawCustom`** hooks per tick.

Types: **`Frame`** in **`lib/Canvas/utils/types/index.ts`**. Implementation: **`ApexPainter.animate`** (**`lib/Canvas/ApexPainter.ts`**).

---

## Signature

```typescript
animate(
  frames: Frame[],
  defaultDuration: number,
  defaultWidth: number = 800,
  defaultHeight: number = 600,
  options?: {
    gif?: boolean;
    gifPath?: string;
    onStart?: () => void;
    onFrame?: (index: number) => void;
    onEnd?: () => void;
  }
): Promise<Buffer[] | undefined>;
```

| Argument | Role |
|----------|------|
| **`frames`** | Non-empty **`Frame[]`** — each entry describes **one** rendered frame. |
| **`defaultDuration`** | Fallback frame duration **in milliseconds** when **`frame.duration`** is omitted. |
| **`defaultWidth`**, **`defaultHeight`** | Canvas size when **`frame.width`** / **`frame.height`** are omitted. |
| **`options.gif`** | **`true`** → encode to disk (**requires **`gifPath`**). |
| **`options.gifPath`** | Destination **`.gif`** file path when **`gif`** is enabled. |
| **`options.onStart`**, **`onFrame`**, **`onEnd`** | Lifecycle callbacks (see below). |

---

## Return value

| Mode | Return |
|------|--------|
| **`options.gif`** falsy | **`Promise<Buffer[]>`** — one **PNG** **`Buffer`** per frame (**`canvas.toBuffer('image/png')`**). |
| **`options.gif === true`** | **`Promise<undefined>`** — output is **only** the file at **`gifPath`** (encoder **`finish`**). |

---

## `Frame` object

Each frame is drawn **in order**: background → optional **`source`** image → **`onDrawCustom`**.

| Field | Effect |
|-------|--------|
| **`width`**, **`height`** | Override **`defaultWidth`** / **`defaultHeight`** for **this** frame only. |
| **`duration`** | Hold time **ms** for this frame (**GIF delay** + internal pacing); defaults to **`defaultDuration`**. |
| **`backgroundColor`** | Solid **`fillRect`** when no gradient/pattern wins. |
| **`gradient`** | **`GradientConfig`** — **`animate`** builds **`linear`** and **`radial`** gradients only; for **`conic`** or extra control use **`onDrawCustom`**. |
| **`pattern`** | **`createPattern`** from **`pattern.source`** (`repeat` optional). |
| **`source`** | Full-frame **`drawImage`** — blend via **`blendMode`** (**`GlobalCompositeOperation`**, default **`source-over`**). |
| **`transformations`** | Applied **before** drawing **`source`**: **`translate`**, **`rotate`** (degrees), **`scaleX`** / **`scaleY`**. Wrapped in **`save`/`restore`**. |
| **`onDrawCustom(ctx, canvas)`** | Extra drawing **after** bitmap **`source`** — receives **`SKRSContext2D`** and **`Canvas`**. |

**Paint precedence:** if **`gradient`** is set it becomes **`fillStyle`**; else **`pattern`**; else **`backgroundColor`**. Then **`fillRect`**. Then **`source`** image if present.

---

## Callbacks & pacing

| Callback | When |
|----------|------|
| **`onStart`** | Before the first frame is rendered. |
| **`onFrame(index)`** | After each frame is encoded (**index** **`0 … frames.length - 1`**). |
| **`onEnd`** | After all frames (and **`encoder.finish()`** when **`gif`**). |

**Important:** after each frame the implementation **`await`s `setTimeout(..., frame.duration || defaultDuration)`** — so total runtime includes **real-time waits** proportional to configured durations (useful for preview pacing; use **`0`** durations carefully if you want fastest generation).

---

## GIF encoding branch

When **`options.gif`** is **`true`**:

1. **`GIFEncoder`** is constructed at **`defaultWidth` × `defaultHeight`** (does **not** resize per odd-sized frames — prefer consistent **`width`/`height`** when encoding GIF).
2. **`encoder.setRepeat(0)`**, **`setQuality(10)`** (library defaults in source).
3. Each frame: **`encoder.setDelay(duration)`**, **`encoder.addFrame(ctx)`** (canvas context after draw).

If **`gif`** is **`true`** but **`gifPath`** is missing, **`animate`** throws.

## Example — PNG sequence (spritesheet source images)

```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const frames = [
  { duration: 80, backgroundColor: '#0f172a', source: './sprite/frame1.png' },
  { duration: 80, backgroundColor: '#0f172a', source: './sprite/frame2.png' },
  { duration: 120, backgroundColor: '#0f172a', source: './sprite/frame3.png' },
];

const pngs = await painter.animate(frames, 100, 256, 256, {
  onFrame: (i) => console.log('done frame', i),
});

pngs?.forEach((buf, i) => fs.writeFileSync(`out-${i}.png`, buf));
```

## Example — custom draw + radial gradient

```typescript
const buffers = await painter.animate(
  [
    {
      duration: 200,
      gradient: {
        type: 'radial',
        startX: 200,
        startY: 200,
        startRadius: 0,
        endX: 200,
        endY: 200,
        endRadius: 280,
        colors: [
          { stop: 0, color: '#38bdf8' },
          { stop: 1, color: '#0f172a' },
        ],
      },
      onDrawCustom(ctx) {
        ctx.fillStyle = '#fff';
        ctx.font = 'bold 48px sans-serif';
        ctx.fillText('Hi', 160, 220);
      },
    },
    {
      duration: 200,
      backgroundColor: '#111827',
      onDrawCustom(ctx) {
        ctx.strokeStyle = '#f472b6';
        ctx.lineWidth = 4;
        ctx.strokeRect(40, 40, 320, 320);
      },
    },
  ],
  150,
  400,
  400
);
```

## Example — write animated GIF

```typescript
await painter.animate(
  [
    { backgroundColor: '#000', duration: 100 },
    { backgroundColor: '#1e293b', duration: 100 },
    { backgroundColor: '#334155', duration: 100 },
  ],
  100,
  320,
  180,
  {
    gif: true,
    gifPath: './out/sequence.gif',
    onEnd: () => console.log('GIF closed'),
  }
);
// Returns undefined — open ./out/sequence.gif
```

## When to prefer **`createGIF`**

| Use **`animate`** | Use **`createGIF`** |
|-------------------|---------------------|
| Quick procedural scenes + optional GIF file | Production GIFs with **per-frame buffers**, **watermarks**, **matte transparency**, **`dispose`**, **`onStart`** streaming |
| Custom **`onDrawCustom`** each tick | **`GIFInputFrame`** pipeline + **`GIFOptions`** |
