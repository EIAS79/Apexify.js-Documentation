# `animate` — declarative frame sequences {#08-animate-api}


**`await painter.animate(frames, defaultDuration, defaultWidth?, defaultHeight?, options?, painterOpts?)`** renders **each** **`Frame`** on its own canvas, optionally sleeps between ticks, and either collects **PNG** **`Buffer`s** **or** streams frames into **`gifPath`** via **`gifencoder`**.

This path is **not** **`createGIF`**: no per-frame **`dispose`**, **`transparentColor`**, **`GIFWatermarkSpec`**, **`onStart`** streaming, or **`skipResizeWhenDimensionsMatch`**. Use **`createGIF`** when you need GIF89a disposal/transparency semantics or Apexify watermark overlays.

Implementation: **`animateFrames`** (**`lib-next/gif/animate-frames.ts`**). **`Frame`** types live in **`types/gif.ts`**.

---

## Signature & validation

```typescript
animate(
  frames: Frame[],
  defaultDuration: number,
  defaultWidth?: number,   // default 800
  defaultHeight?: number,  // default 600
  options?: AnimateOptions,
  painterOpts?: PainterAssetRefsOptions
): Promise<Buffer[] | undefined>;
```

**`validateAnimateInputs`** throws when:

| Condition | Error |
|-----------|--------|
| **`frames`** missing / empty | **`animate: frames array with at least one frame is required.`** |
| **`defaultDuration < 0`** | **`defaultDuration must be a non-negative number`** |
| **`defaultWidth`/`height` ≤ 0** | must be **positive** |
| **`options.gif`** truthy **without** **`gifPath`** | **`gifPath is required when gif is enabled`** |

Optional **`painterOpts`**: when **`resolveAssetRefs: true`**, **`frames`** are deep-resolved (**`maybeResolveRefs`**); **`AnimateOptions`** is processed with **`prepareForRender`** **only** when **`options`** is passed (**same **`resolveAssetRefsDeep`** as **`createGIF`** **`GIFOptions`**).

---

## `AnimateOptions`

```typescript
type AnimateOptions = {
  gif?: boolean;
  gifPath?: string;        // required when gif === true — filesystem path for output .gif
  onStart?: () => void;
  onFrame?: (index: number) => void;
  onEnd?: () => void;
};
```

Hooks fire **around** the loop: **`onStart`** once before iteration; **`onFrame(i)`** after each frame’s **`toBuffer`** **and** optional **`encoder.addFrame`**; **`onEnd`** once after the loop (**preceded by **`encoder.finish()`** when **`gif: true`**).

---

## Return value

| **`options.gif`** | Return |
|-------------------|--------|
| **`false`** / omitted | **`Buffer[]`** — one **PNG** per frame (**`canvas.toBuffer('image/png')`**). |
| **`true`** | **`undefined`** — GIF bytes written **only** to **`gifPath`** (PNG buffers are still produced internally but **not** returned). |

---

## Per-frame rendering order (`Frame`)

Each iteration uses **`frame.width ?? defaultWidth`**, **`frame.height ?? defaultHeight`**:

1. **`clearRect`** full canvas.
2. **`transformations`** (**optional**) — **`save` → `translate` → `rotate` → `scale`** (**degrees** for **`rotate`**).
3. Background paint (**first match wins building **`fillStyle`**):
   - **`gradient`** — **`GradientConfig`** — **only `linear` and `radial`** branches exist (**`conic`** on the type is **ignored** today).
   - else **`pattern`** — **`loadImage(pattern.source)`**, **`createPattern`** (**`repeat`** default **`repeat`**).
   - else **`backgroundColor`** — flat **`fillRect`**.
4. **`source`** (**optional**) — **`loadImage`**, **`globalCompositeOperation`** (**default **`source-over`**), **`drawImage`** full bounds.
5. **`onDrawCustom(ctx, canvas)`** — imperative drawing hook.
6. **`restore`** if transformations were applied.
7. Push **`canvas.toBuffer('image/png')`** to **`buffers`**.
8. GIF branch:** **`encoder.setDelay(frame.duration ?? defaultDuration)`**, **`encoder.addFrame(ctx)`**.
9. **`await sleep(frame.duration ?? defaultDuration)`** — **`setTimeout`** pacing (**milliseconds**) **after every frame**.

Fixed GIF encoder settings when **`gif: true`**: **`setRepeat(0)`**, **`setQuality(10)`** (no API exposure).

---

## `Frame` fields reference

| Field | Role |
|-------|------|
| **`width`**, **`height`** | Override **`defaultWidth`/`Height`** for **this** tick only. |
| **`duration`** | Frame delay (**GIF delay** **and** **`setTimeout`** sleep); falls back to **`defaultDuration`**. |
| **`backgroundColor`** | Solid backdrop. |
| **`gradient`** | **`GradientConfig`** (**linear**/radial implemented). |
| **`pattern`** | **`{ source, repeat? }`** tiled fill. |
| **`source`** | Full-frame raster (**path**, URL, buffer-friendly **`loadImage`** input). |
| **`blendMode`** | **`GlobalCompositeOperation`** when drawing **`source`**. |
| **`transformations`** | **`scaleX/Y`, `rotate`, `translateX/Y`**. |
| **`onDrawCustom`** | Receive **`SKRSContext2D`** + **`Canvas`** for extra vectors/text/shapes. |

## Example — PNG strip only

```typescript
const pngs = await painter.animate(
  [
    { backgroundColor: '#1e293b' },
    {
      gradient: {
        type: 'linear',
        startX: 0,
        startY: 0,
        endX: 400,
        endY: 300,
        colors: [
          { stop: 0, color: '#6366f1' },
          { stop: 1, color: '#ec4899' },
        ],
      },
      duration: 150,
    },
    {
      backgroundColor: '#0f172a',
      source: './logo.png',
      blendMode: 'screen',
      duration: 200,
    },
  ],
  120,
  480,
  320
);
// pngs is Buffer[] length 3
```

## Example — GIF file output

```typescript
await painter.animate(
  [
    {
      backgroundColor: '#020617',
      onDrawCustom: (ctx, canvas) => {
        ctx.fillStyle = '#38bdf8';
        ctx.font = 'bold 48px Arial';
        ctx.fillText('Tick', canvas.width / 2 - 60, canvas.height / 2);
      },
      duration: 180,
    },
    {
      backgroundColor: '#0f172a',
      onDrawCustom: (ctx, canvas) => {
        ctx.fillStyle = '#f472b6';
        ctx.font = 'bold 48px Arial';
        ctx.fillText('Tock', canvas.width / 2 - 60, canvas.height / 2);
      },
      duration: 180,
    },
  ],
  200,
  640,
  360,
  {
    gif: true,
    gifPath: './out/tick-tock.gif',
    onFrame: (i) => console.log('encoded frame', i),
    onEnd: () => console.log('gif closed'),
  }
);
// resolves undefined — open ./out/tick-tock.gif
```

## Choosing **`animate`** vs **`createGIF`**

| Need | Prefer |
|------|--------|
| Chroma transparency / disposal codes | **`createGIF`** |
| Watermarks / **`textOverlay`** on every encoded frame | **`createGIF`** |
| **`onStart`** streaming / huge batch sizes | **`createGIF`** |
| Quick procedural **`Frame`** defs + optional GIF sidecar | **`animate`** |
| Already have **`Buffer`** PNG/JPEG frames | **`createGIF`** + **`GIFInputFrame`** |
| Encode frames to MP4 | **`createVideo({ createFromFrames })`** or [Scene → video](/docs#08-render-scene-to-video) |
| MP4 + trim / captions / SFX | [Video pipeline (advanced)](/docs#01-video-pipeline) |
