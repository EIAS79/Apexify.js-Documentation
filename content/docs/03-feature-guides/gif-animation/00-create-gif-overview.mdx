# GIF & animation (`createGIF`) — overview {#00-create-gif-overview}

**`await painter.createGIF(gifFrames, options)`** encodes animated GIFs via **`gifencoder`**, combining raster frames (buffers, paths, URLs), optional watermarks, chroma transparency, and programmatic **`onStart`** / **`onEnd`** hooks.

Back: [Feature guides hub](/docs#feature-guides-hub)

---

## Signature

```typescript
await painter.createGIF(
  gifFrames: GIFInputFrame[] | undefined,
  options: GIFOptions
);
```

| Mode | **`gifFrames`** | **`options`** |
|------|-----------------|---------------|
| **Static list** | Non-empty array | No **`onStart`** (or **`onStart`** absent). |
| **Programmatic** | **`undefined`** or ignored | **`onStart`** returns **`GIFEncodedFrame[]`** or **`AsyncIterable<GIFEncodedFrame>`**. |

When **`onStart`** is set, it **replaces** the **`gifFrames`** argument entirely.

---

## Return type (depends on **`outputFormat`**)

| **`outputFormat`** | Resolved return |
|--------------------|-----------------|
| **`file`** | **`undefined`** (GIF written to **`outputFile`**). With **`onEnd`**, may return **only** the **`static`** **`Buffer`** — see **[Callbacks & results](/docs#06-create-gif-onend-results)**. |
| **`buffer`** | **`Buffer`** (GIF bytes), or **`{ gif, static }`** when **`onEnd`** returns a buffer. |
| **`base64`** | **`string`**, or paired object as above. |
| **`attachment`** | **`[{ attachment: ReadableStream, name }]`** — **`onEnd`** is **not** invoked on this path today. |

Types: **`GIFResults`**, **`GIFOptions`** in **`lib/Canvas/utils/types/index.ts`**. Implementation: **`GIFCreator`** (`lib/Canvas/extended/GIFCreator.ts`). **`ApexPainter`** wires **`gifCreator.setPainter(this)`** so **`onStart`/`onEnd`** receive the painter instance.
