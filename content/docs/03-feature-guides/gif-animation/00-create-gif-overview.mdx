# GIF & animation — overview {#00-create-gif-overview}

**`await painter.createGIF(gifFrames, options[, painterOpts])`** encodes animated GIFs with **`gifencoder`**, compositing raster frames (buffers, filesystem paths, **`http`** URLs), optional watermarks, chroma transparency, disposal codes, and programmatic **`onStart`** / **`onEnd`** hooks.

**`await painter.animate(frames, defaultDuration, defaultWidth?, defaultHeight?, options?, painterOpts?)`** builds **each** frame on a fresh canvas from declarative **`Frame`** definitions — PNG **`Buffer[]`** **or** an optional GIF file (**see [Animate](/docs#08-animate-api)**).

Implementation: **`lib-next/gif/gif-creator.ts`**, **`animate-frames.ts`**, **`types/gif.ts`**. **`GIFCreator.setPainter`** wires **`onStart`/`onEnd`** second argument to the **`ApexPainter`** instance (**`main.ts`**).

---

## Signatures

```typescript
await painter.createGIF(gifFrames: GIFInputFrame[] | undefined, options: GIFOptions, painterOpts?);

await painter.animate(
  frames: Frame[],
  defaultDuration: number,
  defaultWidth?: number,   // default 800
  defaultHeight?: number, // default 600
  options?: AnimateOptions,
  painterOpts?
);
```

Optional **`painterOpts`** (**`resolveAssetRefs`**, **`prepareForRender`**) resolves **`$…`** placeholders via **`resolveAssetRefsDeep`**. **`createGIF`** runs **`maybeResolveRefs`** on **`gifFrames`** (when provided) **and** **`GIFOptions`** when **`resolveAssetRefs: true`**. **`animate`** resolves **`frames`** the same way; **`AnimateOptions`** is wrapped with **`prepareForRender`** **only** when **`resolveAssetRefs: true`** **and** **`options`** is passed (**no-op** if you omit **`options`** — call **`prepareForRender`** yourself **first** when you need refs inside **`AnimateOptions`**).

## Modes (`createGIF`)

| Mode | **`gifFrames`** | **`options`** |
|------|-----------------|---------------|
| **Static list** | Non-empty **`GIFInputFrame[]`** | Omit **`onStart`**. |
| **Programmatic** | **`undefined`** (ignored if **`onStart`** present) | **`onStart`** yields **`GIFEncodedFrame[]`** or **`AsyncIterable<GIFEncodedFrame>`**. |

## Types (`apexify.js`)

| Type | Role |
|------|------|
| **`GIFInputFrame`** | Static encoder input (**`duration`** + **`buffer`** / **`background`**). |
| **`GIFEncodedFrame`** | **`onStart`** emission shape (**`buffer`** required). |
| **`GIFOptions`** | Encoder canvas, **`repeat`**, **`quality`**, outputs, overlays, **`onStart`/`onEnd`**. |
| **`GIFResults`** | Structured **`buffer`** / **`base64`** / **`attachment`** (partial — **`createGIF`** may return plain **`Buffer`**). |
| **`Frame`** | **`animate`** declarative scene (**background**, **`source`**, transforms, **`onDrawCustom`**). |
| **`GIFDisposalMethod`** | **`0 \| 1 \| 2 \| 3`** disposal codes. |

**`GIFOptions.basDir`** exists on the type but is **unused** by **`GIFCreator`** today.
