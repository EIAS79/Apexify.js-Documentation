# Lines & connectors — overview {#00-custom-lines-overview}

Apexify exposes **two** complementary APIs:

| API | Entry | Best for |
|-----|-------|----------|
| **`createCustom`** | **`await painter.createCustom(options, buffer)`** | Styled segments & connectors: gradients, dashes, arrows, smooth paths, textures (`CustomOptions`). |
| **Path2D pipeline** | **`painter.createPath2D(commands)`**, **`drawPath`**, hit helpers | Explicit geometry from **`PathCommand[]`** (primitives, rounded rects, stars), stroke/fill/transform per **`drawPath`**. |
