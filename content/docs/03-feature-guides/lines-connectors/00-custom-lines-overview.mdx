# Lines & connectors — overview {#00-custom-lines-overview}

Apexify exposes **two** complementary APIs on **`lib-next`** **`ApexPainter`**:

| API | Entry | Best for |
|-----|-------|----------|
| **Custom lines** | **`await painter.path2d.custom(options, buffer)`** | Styled segments & connectors: gradients, dashes, arrows, smooth paths, textures (**`CustomOptions`**). Same behaviour as the historical **`createCustom`** name. |
| **Path2D pipeline** | **`painter.path2d.create`**, **`await painter.path2d.draw`**, **`await painter.detect.*`** | Explicit geometry from **`PathCommand[]`** (primitives, rounded rects, stars), stroke/fill/transform per **`draw`**. |
