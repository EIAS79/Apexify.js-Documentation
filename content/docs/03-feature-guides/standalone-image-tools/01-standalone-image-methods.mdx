# Standalone image APIs — `painter.image` {#01-standalone-image-methods}

Back: [Standalone overview](/docs#00-standalone-image-tools-overview)

**Entry point:** **`painter.image`** — type **`PainterImageUtils`** (**`lib-next/apex-painter/public-types.ts`**), implementation **`painterImageUtils`** (**`lib-next/image/painter-image-utils.ts`**). **`ApexPainter`** attaches it in **`lib-next/apex-painter/main.ts`** (`readonly image: PainterImageUtils`).

Sources are typically **`string`** paths (resolved vs **`process.cwd()`** where the helper reads the filesystem), **HTTP(S) URLs**, or **`Buffer`**, as noted per method.

**Types:** **`ResizeOptions`** (**`lib-next/types/video.ts`**), **`cropOptions`** (**`lib-next/types/common.ts`**), **`ImageFilter`**, **`MaskOptions`**, **`ImageBlendLayer`**, **`BlendOptions`** (**`lib-next/types/image.ts`**), **`StitchOptions`**, **`CollageLayout`**, **`CompressionOptions`**, **`PaletteOptions`** (**`lib-next/types/batch.ts`**), **`GradientConfig`** (**`lib-next/types/common.ts`**).

Unless noted, methods return **`Promise<Buffer>`** (or the documented union). Use **`await painter.image.<method>(…)`**.

---

## Layout, re-encode & palette

| Method | Summary |
|--------|---------|
| **`stitchImages(images, options?)`** | **`images`**: **`string \| Buffer`[]**. **`StitchOptions`**: **`direction`**, **`spacing`**, **`overlap`**, **`blend`**, … · **`lib-next/output/stitch.ts`** |
| **`createCollage(images, layout)`** | **`images`**: **`{ source, width?, height? }[]`**. **`CollageLayout`**: **`type`**, **`columns`**, **`rows`**, **`spacing`**, **`background`**, … |
| **`compress(image, options?)`** | **`CompressionOptions`**: **`format`**, **`quality`**, optional **`maxWidth`** / **`maxHeight`**, … · **`lib-next/output/compression.ts`** |
| **`extractPalette(image, options?)`** | Dominant colours · **`PaletteOptions`**: **`count`**, **`method`**, **`format`** |
| **`validHex(hexColor)`** | **Synchronous** · validates **`#RRGGBB`** (throws if invalid) |

Narrative examples for stitch / collage / compress also appear under [Raster utilities](/docs#03-batch-stitch-compress-output).

## Resize & format conversion

| Method | Summary |
|--------|---------|
| **`resize(resizeOptions)`** | **`ResizeOptions`**: **`imagePath`** (required), **`size`**, **`maintainAspectRatio`**, **`quality`**, **`outputFormat`** (**`png` \| `jpeg`**) · **`resizingImg`** |
| **`imgConverter(source, newExtension)`** | **`source`**: file path string. **`newExtension`**: **`jpeg`**, **`png`**, **`webp`**, **`tiff`**, **`gif`**, **`avif`**, **`heif`**, **`raw`**, **`pdf`**, **`svg`** (case-insensitive) · **`converter`** |

```typescript
const webp = await painter.image.imgConverter("./export/me.png", "webp");

const thumb = await painter.image.resize({
  imagePath: "./in.png",
  size: { width: 400, height: 400 },
  maintainAspectRatio: true,
  quality: 92,
  outputFormat: "jpeg",
});
```

## Convolution-style filters (standalone)

| Method | Summary |
|--------|---------|
| **`effects(source, filters)`** | **`ImageFilter[]`** pipeline (Sharp-style; same family as **`createImage`** filters). **`filters`** must be a non-empty array |

See **`ImageFilter`** in **`lib-next/types/image.ts`** for **`type`**, **`intensity`**, **`value`**, etc.

```typescript
const filtered = await painter.image.effects("./photo.jpg", [
  { type: "blur", intensity: 0.4 },
]);
```

## Colour overlay & analysis

| Method | Summary |
|--------|---------|
| **`colorsFilter(source, filterColor, opacity?)`** | Tint / overlay — **`filterColor`** as **`string`** or **`GradientConfig`** · **`opacity`** **`0…1`** (default **`1`**) |
| **`colorAnalysis(source)`** | Dominant colours · **`{ color, frequency }[]`** (**`frequency`** as string share in implementation) |
| **`colorsRemover(source, colorToRemove)`** | **`colorToRemove`**: **`{ red, green, blue }`** **`0…255`** · matching pixels transparent · may return **`undefined`** on failure |

```typescript
const palette = await painter.image.colorAnalysis("https://example.com/poster.jpg");

const keyed = await painter.image.colorsRemover("./sprite.png", {
  red: 255,
  green: 0,
  blue: 255,
});
```

## Composite blending

| Method | Summary |
|--------|---------|
| **`blend(layers, baseImageBuffer, defaultBlendMode?)`** | Rasterises **`baseImageBuffer`**, then each **`layers[]`** entry: **`image`** (**`string \| Buffer`**, etc.), **`blendMode`**, optional **`position`**, **`opacity`** **`0…1`**. Default canvas composite: **`source-over`**. Returns **`Promise<Buffer>`** (PNG) |

```typescript
const composed = await painter.image.blend(
  [
    { image: "./overlay.png", blendMode: "multiply", opacity: 0.85, position: { x: 0, y: 0 } },
  ],
  basePngBuffer,
  "source-over"
);
```

## Polygon crop

| Method | Summary |
|--------|---------|
| **`cropImage(options)`** | **`cropOptions`** (**`lib-next/types/common.ts`**): **`imageSource`** (**`string`**), **`coordinates`** — **≥ 3** entries **`{ from: {x,y}, to: {x,y}, tension? }`**, **`crop`**: **`inner`** \| **`outer`**, **`radius`**: **`number`** \| **`"circular"`** |

```typescript
const clipped = await painter.image.cropImage({
  imageSource: "./photo.jpg",
  crop: "inner",
  radius: 8,
  coordinates: [
    { from: { x: 0, y: 0 }, to: { x: 400, y: 0 } },
    { from: { x: 400, y: 0 }, to: { x: 400, y: 300 } },
    { from: { x: 400, y: 300 }, to: { x: 0, y: 300 } },
    { from: { x: 0, y: 300 }, to: { x: 0, y: 0 } },
  ],
});
```

## Mask & gradient blend

| Method | Summary |
|--------|---------|
| **`masking(source, maskSource, options?)`** | **`source`** / **`maskSource`**: path, **`Buffer`**, **`PathLike`**, or **`Uint8Array`**. **`MaskOptions`**: **`type`** **`alpha`** \| **`grayscale`** \| **`color`** ( **`color`** requires **`colorKey`** **`#RRGGBB`** ), **`threshold`**, **`invert`**, … |
| **`gradientBlend(source, options)`** | Overlays **`linear` \| `radial` \| `conic`** gradient via **`BlendOptions`** (stops, **`blendMode`**, optional **`maskSource`**, …) |

## Quick reference (copy-paste)

```typescript
import { ApexPainter } from "apexify.js";

const painter = new ApexPainter();

const strip = await painter.image.stitchImages(["./a.png", "./b.png"], {
  direction: "horizontal",
  spacing: 4,
});

const small = await painter.image.compress(strip, { format: "webp", quality: 82 });

painter.image.validHex("#336699"); // throws if malformed

const stats = await painter.image.colorAnalysis("https://example.com/poster.jpg");
```

## `chain` integration

**`painter.chain`** resolves dotted methods, so you can call **`image.resize`**, **`image.stitchImages`**, etc., and pass **`"current"`** for the rolling buffer. See [Raster utilities](/docs#03-batch-stitch-compress-output).
