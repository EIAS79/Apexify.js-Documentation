# Standalone image tools {#00-standalone-image-tools-overview}

Back: [Feature guides hub](/docs#feature-guides-hub)

These **`ApexPainter`** methods operate on **whole images** (paths, URLs, or **`Buffer`s**) via **`lib/Canvas/utils/general/`** (**Sharp**, loaders, helpers). They are **not** the same as layering **`createImage`** on a canvas — use them when you want **resize**, **crop**, **palette analysis**, **removal**, **composite blending**, etc. **outside** a scene graph.
