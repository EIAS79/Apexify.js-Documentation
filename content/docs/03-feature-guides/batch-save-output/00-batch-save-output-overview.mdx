## Mental model

| Flow | APIs |
|------|------|
| **PNG `Buffer` → file** | **`save`**, **`saveMultiple`** |
| **`Buffer` → base64 / data URL / …** | Constructor **`OutputFormat`** + **`outPut`** |
| **Many independent renders** | **`batch`** |
| **Sequential pipe (`current` buffer)** | **`chain`** |

Standalone image utilities (**resize**, **crop**, **color analysis**, …) live in **[Standalone image tools](/docs#00-standalone-image-tools-overview)** — different from **`createImage`** compositing.
