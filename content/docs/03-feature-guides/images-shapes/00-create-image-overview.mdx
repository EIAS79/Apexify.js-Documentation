# Images & shapes (`createImage`) — overview {#00-create-image-overview}

**`await painter.createImage(images, canvasBuffer, options?)`** draws **bitmaps** (paths, URLs, buffers) or **vector shapes** (rectangle, circle, star, …) onto an **existing** canvas produced by **`createCanvas`** (or any **`Buffer`** / **`CanvasResults`** with compatible dimensions).


---

## What you pass in

| Argument | Role |
|----------|------|
| **`images`** | One **`ImageProperties`** object **or** an array (drawn in order). |
| **`canvasBuffer`** | **`Buffer`** PNG from **`createCanvas`**, or **`{ buffer }`** (`CanvasResults`). |
| **`options`** | Optional **`CreateImageOptions`** — **`isGrouped`** + **`groupTransform`** for treating several layers as one transformed unit. |

Return value: **`Promise<Buffer>`** — full canvas PNG after compositing.

---

## Mental model

1. Build a **base** with **`createCanvas`** (size + backgrounds).
2. Call **`createImage`** to **composite** one or more layers on top (each layer = **`ImageProperties`**).
3. Chain **`createText`**, more **`createImage`**, **`save`**, etc., as needed.

**Whole-file resize / crop / stitch outside compositing:** **[Raster, batch & output](/docs#00-raster-batch-output-overview)** — **`painter.image`** (**chapters [04](/docs#04-image-stitch-and-collage)–[07](/docs#07-image-blend-mask-crop-gradient)**).
