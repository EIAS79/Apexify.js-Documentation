# Video & FFmpeg (`createVideo`) — overview {#00-video-overview}

**`await painter.createVideo(options)`** is the single entry point for FFmpeg-backed media operations: probe metadata, extract frames, transcode, trim, filters, audio mixes, compositing, and more.

Back: [Feature guides hub](/docs#feature-guides-hub)

**Canonical types:** **`lib/Canvas/extended/VideoCreator.ts`** — **`VideoCreationOptions`**, **`MixAudioOperation`**, **`MixAudioOverlayClip`**.

**Implementation:** **`VideoCreator`** delegates to **`VideoHelpers`** (`lib/Canvas/utils/Video/videoHelpers.ts`) and **`ApexPainter`** frame helpers (`extractFrames`, `extractAllFrames`, `getVideoInfo`, …).

**Direct API:** **[Imperative video helpers](/docs#08-imperative-video-helpers)** — **`getVideoInfo`**, **`extractFrameAtTime`**, **`extractFrameByNumber`**, **`extractMultipleFrames`**, **`extractFrames`**, **`extractAllFrames`**.

---

## Requirements

- **FFmpeg** and **ffprobe** on **`PATH`**. If missing, **`createVideo`** throws with install hints from **`getFFmpegInstallInstructions`**.
- Optional **vidstab** filters for **`stabilize`** (first pass); FFmpeg falls back to **`hqdn3d`** denoise if analyse/transform fails.
- Per-interval **pitch/speed** in **`adjustVolume`** uses **`rubberband`** — FFmpeg must be built with **librubberband**.

---

## `source`

Every call includes **`source: string | Buffer`**:

| Form | Behaviour |
|------|-----------|
| **Local path** | Resolved vs **`process.cwd()`** when not absolute (same pattern across helpers). |
| **`http(s)` URL** | Downloaded into **`.temp-frames`** for the duration of the command. |
| **`Buffer`** | Written to a temp **`.mp4`** under **`.temp-frames`**, then deleted when safe. |

Some branches **ignore** top-level **`source`** (they only use nested inputs): **`merge`**, **`splitScreen`**, **`batch`**, **`createFromFrames`**. The type still requires **`source`** — pass any placeholder buffer/path if your operation does not use it.

## Shared callback

**`onProgress?: (progress: { percent, time, speed }) => void`** is passed through on **`freezeFrame`**, **`exportPreset`**, **`normalizeAudio`**, **`applyLUT`**, **`addTransition`**, **`addAnimatedText`** (FFmpeg stderr parsing — rough ETA).

## Minimal example

```typescript
import { ApexPainter } from 'apexify.js';
import path from 'path';

const painter = new ApexPainter();

const info = await painter.createVideo({
  source: './input.mp4',
  getInfo: true,
});
console.log(info.duration, info.width, info.height, info.fps);

await painter.createVideo({
  source: './input.mp4',
  trim: {
    startTime: 2,
    endTime: 12,
    outputPath: path.join('out', 'clip.mp4'),
  },
});
```
