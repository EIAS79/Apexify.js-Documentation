# Video options — deep-dive guides {#00-video-options-hub}

These pages explain **what each `createVideo` operation does**, **how every option affects output**, and include **copy-paste examples**. They complement the topic tutorials (**[overview](/docs#00-video-overview)**, metadata, transcode, …).

The old single-page matrix was hard to read — operations are split below by job.

**Rules:** one primary operation per **`createVideo`** call ([router order](/docs#00-video-overview)). **`source`** is always required by TypeScript even when an operation ignores it.
