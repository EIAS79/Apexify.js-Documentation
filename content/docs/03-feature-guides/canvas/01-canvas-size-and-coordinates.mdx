# Canvas size & coordinates {#01-canvas-size-and-coordinates}

Back: [Canvas overview](/docs#00-create-canvas-overview)

---

## `width` and `height`

Pixels of the output PNG.

```typescript
await painter.createCanvas({
  width: 1920,
  height: 1080,
  colorBg: '#000000',
});
```

If you omit both, the creator defaults to **500×500** unless something else sets size first:

- **`customBg.inherit: true`** → size taken from the **image** after load.
- **`videoBg`** (with successful frame extract) → may set size from the frame when **`width`/`height`** omitted.

---

## `x` and `y`

Offset used with **rotation** and layout math (default **0**). Most banners keep **`0`**.

---

## Minimum valid example

```typescript
await painter.createCanvas({
  width: 640,
  height: 360,
  colorBg: '#18181b',
});
```
