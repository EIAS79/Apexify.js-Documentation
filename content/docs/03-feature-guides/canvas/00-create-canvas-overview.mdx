# Canvas & `createCanvas` — overview {#00-create-canvas-overview}

**`await painter.createCanvas(config)`** builds your **first** image: width, height, and everything drawn **behind** later text/images.

---

## What you get back

```typescript
const { buffer, canvas } = await painter.createCanvas({ ... });
```

- **`buffer`** — PNG **`Buffer`** (save or pass to **`createText`** / **`createImage`**).
- **`canvas`** — the **`CanvasConfig`** you passed (echo for debugging).
