# Canvas & `createCanvas` — overview {#00-create-canvas-overview}

**`await painter.createCanvas(config[, painterOpts])`** allocates a **`@napi-rs/canvas`** surface, paints backgrounds (layers, gradients, images, patterns, noise), applies clip, zoom, rotation, blur, optional shadow and stroke, and returns PNG bytes. Most Apexify pipelines start here — **`createImage`**, **`createText`**, and charts composite on top of **`CanvasResults.buffer`** (or a plain **`Buffer`** from elsewhere).

---

## Signature & types

```typescript
await painter.createCanvas(
  config: CanvasConfig,
  painterOpts?: { resolveAssetRefs?: boolean }
): Promise<CanvasResults>;
```

**`CanvasConfig`**, **`BackgroundLayer`**, **`CanvasResults`** — **`apexify.js`** (**`lib-next/types/canvas.ts`**). With **`resolveAssetRefs: true`**, **`$…`** placeholders in **`config`** resolve via **`painter.assets`** (**same **`maybeResolveRefs`** path as **`main.ts`**).

---

## Return value (**`CanvasResults`**)

```typescript
const { buffer, canvas } = await painter.createCanvas({
  width: 800,
  height: 450,
  colorBg: '#0f172a',
});
```

| Field | Meaning |
|-------|---------|
| **`buffer`** | **`image/png`** **`Buffer`** — save, **`outPut`**, or pass to **`createImage`** **`/`** **`createText`**. |
| **`canvas`** | The **`CanvasConfig`** reference you passed (**may be mutated**: **`customBg.inherit`** sets **`width`** **`/`** **`height`** from the image; **`videoBg`** can fill missing dimensions). Useful for inspecting the effective size after those steps. |

---

## Defaults & validation (quick rules)

| Topic | Behaviour |
|-------|-----------|
| **Size** | If **`width`** **`/`** **`height`** stay unset after **`inherit`** **`/`** **`videoBg`** sizing, they default to **`500`** each — prefer explicit dimensions in production. |
| **Primary fill mutex** | At most **one** of **`colorBg`**, **`gradientBg`**, **`customBg`** — supplying more than one throws (**see [Primary backgrounds](/docs#02-backgrounds-primary)**). |
| **`opacity`** | Canvas global alpha must be **`0`** **`–`** **`1`**. |
| **`zoom.scale`** | Must be **`> 0`** when set. |
| **`bgLayers`** | Each layer **`type`** is **`color`**, **`gradient`**, **`image`**, **`pattern`**, **`presetPattern`**, or **`noise`** (**per-entry validation**). |

**`videoBg`**, **`patternBg`**, **`noiseBg`**, **`transparentBase`**, and **`bgLayers`** interact with primary fills as documented in **[Paint order](/docs#09-canvas-paint-order)**.

---

## Minimal example

```typescript
import { ApexPainter } from 'apexify.js';

const painter = new ApexPainter({ type: 'buffer' });

const { buffer } = await painter.createCanvas({
  width: 640,
  height: 360,
  colorBg: '#020617',
});

// buffer → save, createImage, createText, charts…
```
