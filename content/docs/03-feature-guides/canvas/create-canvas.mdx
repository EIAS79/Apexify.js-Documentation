# Create canvas (`createCanvas`)

**API:** `await painter.createCanvas(canvas: CanvasConfig)` → **`CanvasResults`** (`buffer`, dimensions, and helpers depending on config).

**`createCanvas`** allocates an image at **`width`** × **`height`**, paints backgrounds (**solid**, gradients, layers, patterns, noise), and returns a PNG **`buffer`** you pass into **`createText`**, **`createImage`**, and other draws.
