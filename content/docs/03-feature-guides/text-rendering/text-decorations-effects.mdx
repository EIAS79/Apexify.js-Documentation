# Decorations & effects

Underline-style decorations, highlight bands, glow, shadow, outline stroke, and rotation — all on **`TextProperties`**.

Overview: [Create text](/docs#create-text).

---

## **`underline`**, **`overline`**, **`strikethrough`**

Each can be **`true`** (default styling) or an object with **`color`**, **`gradient`**, **`width`**.

<CodeSwitcher>
<ts>
```typescript
await painter.createText(
  {
    text: 'Limited · while supplies last',
    x: 80,
    y: 120,
    font: { size: 30, family: 'Arial' },
    color: '#fefce8',
    underline: { color: '#fbbf24', width: 3 },
    overline: { color: 'rgba(254,243,199,0.45)', width: 2 },
    strikethrough: true,
  },
  baseBuffer
);
```
</ts>
<js>
```javascript
await painter.createText(
  {
    text: 'Limited · while supplies last',
    x: 80,
    y: 120,
    font: { size: 30, family: 'Arial' },
    color: '#fefce8',
    underline: { color: '#fbbf24', width: 3 },
    overline: { color: 'rgba(254,243,199,0.45)', width: 2 },
    strikethrough: true,
  },
  baseBuffer
);
```
</js>
</CodeSwitcher>

On **`textOnCurve`**, decorations follow each glyph along the arc (see [Curved text & metrics](/docs#text-curved-and-metrics)).

```typescript
  },
  baseBuffer
```

```javascript
await painter.createText(
```

Use **`highlight.gradient`** instead of **`color`** for a ramp behind the glyphs.

## **`shadow`**

Supports solid **`color`** or **`gradient`**, offsets, blur, and opacity.

<CodeSwitcher>
<ts>
```typescript
await painter.createText(
  {
    text: 'Depth without Photoshop',
    x: 120,
    y: 180,
    font: { size: 48, family: 'Georgia' },
    color: '#fef3c7',
    shadow: {
      color: 'rgba(76,29,29,0.55)',
      offsetX: 6,
      offsetY: 12,
      blur: 18,
      opacity: 1,
    },
  },
  baseBuffer
```

```typescript
    gradient: {
      type: 'linear',
      startX: 0,
      startY: 0,
```
