# Curved text & metrics

Use **`textOnCurve`** for circular banners, and **`measureText`** to preview widths, wrapped lines, and optional per-character sizes **without** exporting a PNG.

Overview: [Create text](/docs#create-text).

## Measure text (`measureText`)

**API:** `await painter.measureText(textProps)` â†’ **`TextMetrics`**.

- Allocates an **internal** canvas â€” nothing is drawn on your scene buffer.
- Registers custom fonts the same way **`createText`** does when **`font.path`** / **`fontPath`** is set (relative paths join **`process.cwd()`**; failures **warn**).
- With **`maxWidth`**, you receive **`lines[]`**, **`lineCount`**, and **`totalHeight`** matching the wrap helper used by **`EnhancedTextRenderer`**.
- With **`includeCharMetrics: true`**, you also get **`charWidths`** and **`charPositions`** for each glyph.
- With **`measurementCanvas`**, cap the scratch surface **`width`** / **`height`** instead of auto-estimating from string length.

<CodeSwitcher>
<ts>
```typescript
const painter = new ApexPainter();

const metrics = await painter.measureText({
  text: 'Measure me before you reserve layout space.',
  font: { size: 28, family: 'Arial' },
  maxWidth: 420,
  lineHeight: 1.35,
  letterSpacing: 0.5,
});

console.log(metrics.width, metrics.totalHeight, metrics.lineCount);
if (metrics.lines) {
  metrics.lines.forEach((line) => console.log(line.text, line.width));
}
```
</ts>
<js>
```javascript
const painter = new ApexPainter();

(async () => {
  const metrics = await painter.measureText({

    metrics.lines.forEach((line) => console.log(line.text, line.width));
  }
})();
```

### Curved sizing preview

When **`textOnCurve`** is present and **`0 < sweepAngle < 360`**, **`measureText`** adjusts reported **`width`** / **`height`** to approximate the chord / sagitta of the arc â€” use it to reserve bounding boxes before drawing.

<CodeSwitcher>
<ts>
```typescript
const curvedBox = await painter.measureText({
  text: 'CURVED METRICS',
  font: { size: 42, family: 'Arial' },
  textOnCurve: { sweepAngle: 75, radius: 240, up: true },
});

console.log(curvedBox.width, curvedBox.height);
```
</ts>
<js>
```javascript
const painter = new ApexPainter();

(async () => {
```

## What returns on **`TextMetrics`**

Always includes Canvas **`measureText`** fields (**`width`**, bounding boxes, baselines when available) plus Apexify extras **`height`**, **`lineHeight`**, **`baseline`**, **`top`**, **`bottom`**, **`centerX`**, **`centerY`**.

Optional arrays:

- **`lines`** / **`lineCount`** / **`totalHeight`** â€” when **`maxWidth`** is set.
- **`charWidths`** / **`charPositions`** â€” when **`includeCharMetrics`** is **`true`**.

If you already own a **`SKRSContext2D`**, **`CanvasUtils.drawText`** / **`WrappedText`** expose a smaller synchronous subset (no **`textOnCurve`** stack). Reach for **`createText`** when you need the full renderer.
