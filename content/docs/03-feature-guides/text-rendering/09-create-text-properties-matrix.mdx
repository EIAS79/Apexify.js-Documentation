# `TextProperties` — full field matrix {#09-create-text-properties-matrix}

Single-page reference aligned with **`TextProperties`** (**`lib-next/types/text.ts`**).

---

## Required

| Field | Type | Notes |
|-------|------|-------|
| **`text`** | **`string`** | Unicode string (**`\\n`** splits rows). |
| **`x`** | **`number`** | Anchor **X**. |
| **`y`** | **`number`** | Anchor **Y**. |

---

## Font

| Field | Type | Notes |
|-------|------|-------|
| **`font`** | **`{ size?, family?, name?, path? }`** | Preferred grouping |
| **`fontSize`** | **`number`** | Deprecated → **`font.size`** (**default `16`**) |
| **`fontFamily`** | **`string`** | Deprecated → **`font.family`** |
| **`fontName`** | **`string`** | Deprecated → **`font.name`** |
| **`fontPath`** | **`string`** | Deprecated → **`font.path`** |

Resolution order for family string: **`font.name` → `fontName` → `font.family` → `fontFamily` → `"Arial"`**.

---

## Layout (`layout` + legacy mirrors)

| Nested | Legacy alias | Role |
|--------|----------------|------|
| **`layout.lineHeight`** | **`lineHeight`** | Multiplier × **`font.size`** (**default `1.4`** logic when omitted) |
| **`layout.letterSpacing`** | **`letterSpacing`** | Pixels → **`ctx.letterSpacing`** |
| **`layout.wordSpacing`** | **`wordSpacing`** | Pixels → **`ctx.wordSpacing`** |
| **`layout.maxWidth`** | **`maxWidth`** | Enables **`computeWrappedTextLines`** |
| **`layout.maxHeight`** | **`maxHeight`** | Caps wrapped line count |

## Stroke & arc extras

| Field | Type | Role |
|-------|------|------|
| **`stroke`** | **`TextStrokeStyle`** | Outline (**`color`**, **`width`**, **`gradient`**, **`opacity`**, **`style`**) |
| **`textOnCurve`** | **`TextCurveConfig`** | Circular arc layout (**see [Curved text](/docs#06-create-text-curved)**) |

---

## Measurement-only

| Field | Type | Role |
|-------|------|------|
| **`includeCharMetrics`** | **`boolean`** | Adds per-code-unit widths/positions (**`measureText`**) |
| **`measurementCanvas`** | **`{ width?, height? }`** | Scratch surface sizing (**`measureText`**) |

---

## Deprecated interface

**`TextObject`** extends **`TextProperties`** with **`isBold?`**, **`outlined?`** — do not use for new **`createText`** calls.

---

## Merge precedence (summary)

Whenever a property exists both **nested** and **flat**, **nested wins** — see resolver exports at the bottom of **`types/text.ts`**.
