## Font configuration

Prefer the nested **`font`** object; legacy top-level fields remain for compatibility.

| Field | Purpose |
|-------|---------|
| **`font.size`** | Pixels (default **16**). |
| **`font.family`** | Installed family name (e.g. **`Arial`**). |
| **`font.name`** | Logical name used when registering **`font.path`**. |
| **`font.path`** | `.ttf` / `.otf` / `.woff` path — resolved vs **`process.cwd()`** unless absolute. |

Legacy aliases: **`fontSize`**, **`fontFamily`**, **`fontName`**, **`fontPath`** (same precedence as **`font.*`** in **`EnhancedTextRenderer.setupFont`**).

```typescript
await painter.createText(
  {
    text: 'Hello',
    x: 120,
    y: 96,
    font: { size: 42, path: './fonts/Inter-Bold.ttf', name: 'InterDisplay' },
    bold: false,
  },
  buffer
);
```

**`GlobalFonts.registerFromPath`** runs per render when **`path`** set — failures log **`console.warn`** and fall through.

## Weight & slant

| Field | Effect |
|-------|--------|
| **`bold`** / **`decorations.bold`** | Prefixes **`bold`** in CSS font string (nested wins if both set). |
| **`italic`** / **`decorations.italic`** | Prefixes **`italic`**. |

## Alignment, baseline & rotation

Prefer **`placement`**; flat **`textAlign`**, **`textBaseline`**, and **`rotation`** remain aliases (**`resolveTextPlacement`** in **`lib-next/types/text.ts`**).

| Field | Values |
|-------|--------|
| **`textAlign`** | **`'left' \| 'center' \| 'right' \| 'start' \| 'end'`** |
| **`textBaseline`** | **`'alphabetic' \| 'bottom' \| 'hanging' \| 'ideographic' \| 'middle' \| 'top'`** |
| **`rotation`** | Degrees around **`(x, y)`** — applied in **`applyTextTransformations`** before measure/draw. |

```typescript
await painter.createText(
  {
    text: 'Centered',
    x: 320,
    y: 120,
    font: { size: 28, family: 'Arial' },
    placement: { textAlign: 'center', textBaseline: 'middle', rotation: -4 },
  },
  buffer
);
```

## Fill opacity

Overall glyph fill alpha: prefer **`fill.opacity`**, or legacy top-level **`opacity`** (**`resolveTextFill`** → **`text-layout`** sets **`ctx.globalAlpha`** when defined).
