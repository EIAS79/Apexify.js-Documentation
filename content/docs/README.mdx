# Apexify.js documentation {#readme}

This directory is the **human-readable docs** for **Apexify.js**: tutorials, feature guides, API sheets, and contributor notes.

---

## Where to begin

1. **[Start here](/docs#00-start-here)** — what the library is, install, **`ApexPainter`**, minimal example (**`00-start-here/00-start-here.mdx`**).
2. **[Feature tracks](/docs#01-feature-tracks)** — every major topic with a **short summary** and **links** into the right overview (**`00-start-here/01-feature-tracks.mdx`**).
3. **[Beginner guide](/docs#00-beginner-guide)** — first real images and buffers (**`01-beginner-guide/00-beginner-guide.mdx`**).
4. **[What you can build](/docs#01-what-you-can-build)** — idea catalog (products → APIs) across **all** features; then **[Recipes hub](/docs#00-recipes-overview)** for copy-paste **`recipe-*.mdx`** flows as you add them.

Learning flow lives in **`docs/00-start-here/`** and **`docs/01-beginner-guide/`**. Inspiration and goal-oriented pages live under **`docs/02-recipes/`**.
