## TypeScript signatures & types

There is **no separate generated API micro-site** in this repo: behaviour is documented in **`03-feature-guides/`**, and **signatures** live in **`lib-next/`** (published as **`apexify.js`** and **`apexify.js/types`**).

| Looking for | Where |
|-------------|--------|
| **`ApexPainter`** methods | **`lib-next/apex-painter/main.ts`** |
| Exported surface and re-exports | **`lib-next/index.ts`** |
| Public types | **`lib-next/types/`** (package export **`apexify.js/types`**) |
| Domain internals | **`lib-next/canvas/`**, **`gif/`**, **`video/`**, **`chart/`**, … |
