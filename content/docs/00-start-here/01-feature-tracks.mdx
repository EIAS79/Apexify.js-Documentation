## Suggested order if you are learning

1. **[Beginner guide](/docs#00-beginner-guide)** — first PNG, buffers, saving, mistakes.
2. **[Canvas](/docs#00-create-canvas-overview)** — almost everything draws onto a canvas you create first.
3. Pick **one** of Images, Text, or GIF next (whatever matches your project).
4. Add **Lines**, **Charts**, **Video**, or **Pixels** only when you need that capability.
