## Suggested order if you are learning

1. **[Beginner guide](/docs#00-beginner-guide)** — first PNG, buffers, saving, mistakes.
2. **[Canvas](/docs#00-create-canvas-overview)** — almost everything draws onto a canvas you create first.
3. Pick **one** of Images, Text, or GIF next (whatever matches your project).
4. Add **Recipes** (**[philosophy](/docs#02-philosophy-goals-and-when-to-use)**, **[what you can build](/docs#01-what-you-can-build)**) when you need outcome framing **`/`** **`recipe-*`** snippets.
5. Add **Lines**, **Charts**, **[Video](/docs#00-video-overview)**, **[Raster / batch / save](/docs#00-raster-batch-output-overview)**, or **Advanced** ([Composition](/docs#00-composition-hub) · [Scene](/docs#00-scene-composition) · [Video pipeline](/docs#01-video-pipeline) · [`createAudio`](/docs#01-create-audio)) when you need that capability.
