# Start here {#00-start-here}

Welcome to **Apexify.js** — a Node.js library for building **images** (PNGs), **GIFs**, **charts**, and **video helpers** with code instead of a design tool.

**Deep links:** anchors match **`docs/00-start-here/*.mdx`** basenames — **`00-start-here.mdx`** → **`/docs#00-start-here`**, **`01-feature-tracks.mdx`** → **`/docs#01-feature-tracks`**.

---

## Who this is for

- You know basic **JavaScript** or **TypeScript**.
- You want **repeatable** graphics: banners, thumbnails, social posts, batch exports.
- You are okay using **`async`/`await`** (most APIs return **Promises**).

---

## Install

```bash
npm install apexify.js
```

**Optional:** for video backgrounds and **`createVideo`**, install **FFmpeg** and **ffprobe** and ensure they are on your **`PATH`**.

---

## The entry point

Everything flows through **`ApexPainter`**:

```typescript
import { ApexPainter } from 'apexify.js';

const painter = new ApexPainter();
```

## Quick smallest example

```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const { buffer } = await painter.createCanvas({
  width: 640,
  height: 360,
  colorBg: '#0f172a',
});

fs.writeFileSync('out.png', buffer);
```
