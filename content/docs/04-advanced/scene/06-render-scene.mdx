# `renderScene` {#06-render-scene}


**`renderScene(input: SceneRenderInput, options?: SceneRenderOptions): Promise<Buffer>`** — async because chart/image loads are async. Returns **PNG** bytes.

**`SceneCreator.render`** (used internally) accepts the same **`options`**: by default it runs **`validateSceneRenderInput`** (dimensions, **`layers`** array, nested **`surface`** depth). Pass **`{ validate: false }`** only for trusted hot paths; use **`{ maxSurfaceDepth: n }`** to cap nesting (default **64**).

---

## Variant: plain object (config / CMS JSON)

```typescript
import { ApexPainter } from "apexify.js";
import fs from "fs";

const painter = new ApexPainter();

const png = await painter.renderScene({
  width: 1200,
  height: 630,
  background: { colorBg: "#1d4ed8" },
  layers: [
    {
      type: "text",
      texts: [
        { text: "Social card", x: 60, y: 200, fontSize: 64, color: "#ffffff" },
        { text: "apexify.js", x: 60, y: 300, fontSize: 28, color: "#bfdbfe" },
      ],
    },
  ],
});

await fs.promises.writeFile("card.png", png);
```

## Variant: validate before work (untrusted JSON)

```typescript
import { ApexPainter, validateSceneRenderInput } from "apexify.js";
import type { SceneRenderInput } from "apexify.js";

const painter = new ApexPainter();

function parseScene(json: unknown): SceneRenderInput {
  const input = json as SceneRenderInput;
  validateSceneRenderInput(input, { maxSurfaceDepth: 16 });
  return input;
}

// or: painter.validateSceneRenderInput(input, { maxSurfaceDepth: 16 });
```

---

## Variant: skip validation (trusted pipeline)

```typescript
await painter.renderScene(trustedInput, { validate: false });
```

## Named assets (`$name`, **`$palette.key`**)

Strings in **`SceneRenderInput`** normally resolve **`$`** references via **`painter.assets`** (**on** unless **`resolveAssetRefs: false`**). See **[Composition hub](/docs#00-composition-hub)** and **[Imperative & batch resolution](/docs#05-imperative-batch-chain-assets)** for **`prepareForRender`**, imperative APIs, and **`batch`** / **`chain`**.

## Variant: `SceneBuilder` then `render` (`addLayers` + options)

```typescript
const png = await painter
  .createScene(960, 540)
  .setBackground({ colorBg: "#020617" })
  .addLayers([
    {
      type: "path",
      path: [
        { type: "moveTo", x: 48, y: 120 },
        { type: "lineTo", x: 400, y: 120 },
      ],
      options: { stroke: { color: "#38bdf8", width: 4 } },
    },
    {
      type: "text",
      texts: [{ text: "Builder", x: 48, y: 160, fontSize: 48, color: "#f1f5f9" }],
    },
  ])
  .render({ maxSurfaceDepth: 32 });
```

const baseScene = {

## Variant: dynamic layers from a function

```typescript
function buildScene(theme: "light" | "dark") {
  const bg = theme === "dark" ? "#0f172a" : "#f8fafc";
  const fg = theme === "dark" ? "#e2e8f0" : "#0f172a";
  return {
    width: 400,
    height: 200,
    background: { colorBg: bg },
    layers: [{ type: "text" as const, texts: [{ text: theme, x: 24, y: 80, fontSize: 32, color: fg }] }],
  };
}

const darkPng = await painter.renderScene(buildScene("dark"));
const lightPng = await painter.renderScene(buildScene("light"));
```

---

## Error handling

Failures wrap as **`SceneCreator.render failed: …`**. **`validateSceneRenderInput`** throws earlier with **`Scene: …`** messages for bad dimensions or excessive **`surface`** nesting. Validate **`customLines`** and file paths **before** render in production when validation alone is not enough.
