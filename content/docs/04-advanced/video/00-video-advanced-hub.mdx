# Video — advanced workflows {#00-video-advanced-hub}

## When to use which API

| Goal | API |
|------|-----|
| **Video editor**, saved edit JSON, server render farm | **`videoPipeline()`** |
| **One** FFmpeg operation | **`createVideo({ … })`** |
| **Designed in Scene**, then encode | **`renderSceneToVideoFrames`** |
| **Generated** frames (`animate`) | **`animate`** → **`createFromFrames`** or pipeline |
