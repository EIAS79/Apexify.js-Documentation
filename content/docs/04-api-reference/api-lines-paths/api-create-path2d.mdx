# createPath2D()

## Description

Builds a Path2D-like path object from **`PathCommand[]`** for advanced vector workflows paired with **`drawPath`**.

## Syntax

```ts
painter.createPath2D(commands: PathCommand[]): any
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| **`commands`** | **`PathCommand[]`** | Yes | — | Imperative path ops consumed by the internal **`Path2DCreator`**. |

Published typings return **`any`** (native/path handle).

## Returns

Path handle (opaque **`any`** in **`apexify.js` typings).

## Example

```typescript
const path = painter.createPath2D([
  { type: 'moveTo', x: 10, y: 10 },
  { type: 'lineTo', x: 200, y: 140 },
  { type: 'closePath' },
]);
```

*(Exact **`PathCommand`** unions — see **`apexify.js`** typings / [Path2D & drawPath](/docs#path2d-draw).)*

## Common errors

| Error | Cause | Fix |
|-------|--------|-----|
| Unsupported / malformed commands | Invalid **`PathCommand`** sequence | Follow examples in [Path2D & drawPath](/docs#path2d-draw). |
