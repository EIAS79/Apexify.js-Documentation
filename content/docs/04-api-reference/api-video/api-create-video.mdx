# createVideo()

## Description

Single entry router for FFmpeg-backed workflows (**info**, **frame extraction**, **transcode**, filters, concat, audio mixes, …). Exactly **one** operation branch executes per call — see [Video router](/docs#video-create-video).

## Syntax

```ts
await painter.createVideo(options: VideoCreationOptions): Promise<any>
```

Published typings currently return **`Promise<any>`** — the concrete shape depends on which operation keys are set.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| **`options`** | **`VideoCreationOptions`** | Yes | — | Must include **`source`** (**`string`** \| **`Buffer`**) plus **one** primary operation group (`convert`, `getInfo`, `extractFrame`, …). |

## Returns

**Operation-dependent** — infer from the branch you triggered (buffers, paths, metadata objects). Use **`VideoCreationOptions`** typings / [Video router](/docs#video-create-video) tables.

## Example

```typescript
const meta = await painter.createVideo({
  source: './clips/demo.mp4',
  getInfo: true,
});
```

## Common errors

| Error | Cause | Fix |
|-------|--------|-----|
| **`No video operation specified`** | No recognized branch matched | Add exactly one supported operation object. |
| FFmpeg missing / PATH issues | Binaries unavailable | Install FFmpeg + ffprobe — see [Troubleshooting](/docs#troubleshooting). |
