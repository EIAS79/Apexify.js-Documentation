# createCanvas()

## Description

Creates a new raster canvas from **`CanvasConfig`** (size, backgrounds, layers, patterns, noise, optional zoom/shadow/stroke). Returns a **`CanvasResults`** object whose **`buffer`** is PNG data you pass into later draws.

## Syntax

```ts
await painter.createCanvas(canvas: CanvasConfig): Promise<CanvasResults>
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| **`canvas`** | **`CanvasConfig`** | Yes | — | Width/height, fills (**`colorBg`**, **`gradientBg`**, **`customBg`**, **`videoBg`**), **`bgLayers`**, **`patternBg`**, **`noiseBg`**, **`transparentBase`**, transforms, optional border/shadow. |

### Important options

| Option | Type | Description |
|--------|------|-------------|
| **`width`**, **`height`** | **`number`** | Scene size in pixels. |
| **`colorBg`** | **`string`** | Solid fill (CSS-like color). |
| **`gradientBg`**, **`customBg`**, **`bgLayers`** | See **`CanvasTypes`** | Rich backgrounds — see feature guides. |

Full shape: **`CanvasTypes.CanvasConfig`** from **`apexify.js`**.

## Returns

**`CanvasResults`** (published typings):

| Field | Type | Description |
|-------|------|-------------|
| **`buffer`** | **`Buffer`** | PNG raster. |
| **`canvas`** | **`CanvasConfig`** | Round-tripped / normalized config snapshot (see typings). |

## Example

```typescript
const { buffer } = await painter.createCanvas({
  width: 800,
  height: 450,
  colorBg: '#111827',
});
```

## Common errors

| Error | Cause | Fix |
|-------|--------|-----|
| Canvas validation errors | Conflicting **`CanvasConfig`** (sizes, media paths, layers) | Follow [Canvas pipeline](/docs#canvas-pipeline) validation rules; verify paths/URLs. |
