# createComparisonChart()

## Description

Renders **two** charts on one PNG (**side-by-side** or **stacked**) with independent panel configs (**`chart1`**, **`chart2`**), including **`donut`** panels only available here (not via **`createChart('pie')`** alone).

## Syntax

```ts
await painter.createComparisonChart(
  options: ComparisonChartOptions,
): Promise<Buffer>
```

**`ComparisonChartOptions`** is declared in the library’s Charts comparison module (import path varies by package layout — rely on editor autocomplete from **`apexify.js`** or see feature doc tables).

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| **`options`** | **`ComparisonChartOptions`** | Yes | — | Layout, spacing, dimensions, shared appearance, **`generalTitle`**, **`chart1`**, **`chart2`**. |

See field-level tables in [Charts: comparison & combo](/docs#charts-comparison-combo).

## Returns

PNG **`Buffer`**.

## Example

```typescript
const img = await painter.createComparisonChart({
  layout: 'sideBySide',
  dimensions: { width: 960, height: 420 },
  chart1: {
    type: 'bar',
    title: 'Q1',
    data: [{ label: 'A', value: 12, xStart: 0, xEnd: 1, color: '#38bdf8' }],
    options: { dimensions: { height: 320 } },
  },
  chart2: {
    type: 'line',
    title: 'Trend',
    data: [{ label: 'S', data: [{ x: 1, y: 2 }, { x: 2, y: 5 }], color: '#f97316' }],
    options: {},
  },
});
```

## Common errors

| Error | Cause | Fix |
|-------|--------|-----|
| Panel type/data mismatch | **`ComparisonChartConfig`** doesn’t match **`type`** | Align **`data`**/**`options`** with the panel **`type`**. |
