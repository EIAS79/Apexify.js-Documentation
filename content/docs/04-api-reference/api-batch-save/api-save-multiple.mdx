# saveMultiple()

## Description

Persists an array of **`Buffer`**s using shared **`SaveOptions`** (same naming rules as **`save`**, incrementing counters / timestamps per entry).

## Syntax

```ts
await painter.saveMultiple(buffers: Buffer[], options?: SaveOptions): Promise<SaveResult[]>
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| **`buffers`** | **`Buffer[]`** | Yes | — | Each buffer saved sequentially with shared options. |
| **`options`** | **`SaveOptions`** | No | — | Same defaults as **`save()`** — see [save()](/docs#api-save). |

## Returns

Array of **`SaveResult`** objects aligned with input buffers.

## Example

```typescript
const a = await painter.createCanvas({ width: 120, height: 120, colorBg: '#22c55e' });
const b = await painter.createCanvas({ width: 120, height: 120, colorBg: '#3b82f6' });

await painter.saveMultiple([a.buffer, b.buffer], {
  directory: './renders',
  naming: 'counter',
  prefix: 'slide-',
});
```

## Common errors

| Error | Cause | Fix |
|-------|--------|-----|
| Same as **`save`** | Invalid buffers / FS issues | Ensure each entry is a **`Buffer`**; verify paths. |
