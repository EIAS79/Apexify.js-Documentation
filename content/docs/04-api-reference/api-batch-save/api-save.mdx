# save()

## Description

Writes a PNG (**or Sharp-encoded**) **`Buffer`** to disk with optional naming strategies (**timestamp**, **counter**, **custom**).

## Syntax

```ts
await painter.save(buffer: Buffer, options?: SaveOptions): Promise<SaveResult>
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| **`buffer`** | **`Buffer`** | Yes | — | Raster payload (**PNG** recommended unless **`format`** triggers transcoding). |
| **`options`** | **`SaveOptions`** | No | See runtime defaults below | Directory, filename, **`format`**, **`quality`**, naming helpers. |

### Runtime defaults (**apexify.js 5.3.20**)

| Option | Default | Notes |
|--------|---------|------|
| **`directory`** | **`./ApexPainter_output`** | Verified in implementation (`ApexPainter.save`). Published **`SaveOptions`** typings mention `./output` — **typings mismatch**; trust runtime default above until upstream aligns. |
| **`format`** | **`'png'`** | JPEG/WebP/AVIF run through Sharp. |
| **`naming`** | **`'timestamp'`** | Alternatives: **`counter`**, **`custom`**. |

Other fields match **`SaveOptions`** (**`quality`**, **`prefix`**, **`suffix`**, **`overwrite`**, **`createDirectory`**, …).

## Returns

**`SaveResult`**: **`path`**, **`filename`**, **`size`**, **`format`**.

## Example

```typescript
const { buffer } = await painter.createCanvas({
  width: 400,
  height: 240,
  colorBg: '#111827',
});

await painter.save(buffer, {
  directory: './renders',
  filename: 'card.png',
});
```

## Common errors

| Error | Cause | Fix |
|-------|--------|-----|
| **`save: buffer must be a Buffer.`** | Wrong input type | Pass **`Buffer`** only. |
| Filesystem errors | Missing permissions / invalid path | Ensure **`directory`** writable; enable **`createDirectory`**. |
