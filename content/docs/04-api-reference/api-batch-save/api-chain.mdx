# chain()

## Description

Invokes a sequence of **`ApexPainter`** methods **sequentially**, threading the latest PNG **`Buffer`** between steps. Each **`ChainOperation`** is **`{ method, args }`** where **`method`** is a **string key** of **`ApexPainter`** and **`args`** becomes the argument list.

Special-case placeholders inside **`args`**:

- Literal **`'current'`** — replaced with the latest buffer.
- Objects tagged **`__isCurrentBuffer: true`** — also replaced with the latest buffer.

*(Behavior verified from `apexify.js` `chainOperations` helper implementation.)*

## Syntax

```ts
await painter.chain(operations: ChainOperation[]): Promise<Buffer>
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| **`operations`** | **`ChainOperation[]`** | Yes | — | Non-empty array of **`{ method: string; args: any[] }`**. |

### `ChainOperation`

| Field | Type | Description |
|-------|------|-------------|
| **`method`** | **`string`** | Must exist on **`ApexPainter`** as a function. |
| **`args`** | **`any[]`** | Passed to **`apply`** after **`current`** substitution. |

Each step’s return must be a **`Buffer`** **or** an object with **`buffer`** (e.g. **`CanvasResults`**).

## Returns

Final PNG **`Buffer`**.

## Example

```typescript
const finalBuf = await painter.chain([
  {
    method: 'createCanvas',
    args: [{ width: 640, height: 360, colorBg: '#020617' }],
  },
  {
    method: 'createText',
    args: [
      {
        text: 'Chained',
        x: 320,
        y: 200,
        font: { size: 48, family: 'Arial' },
        color: '#f8fafc',
        textAlign: 'center',
        textBaseline: 'middle',
      },
      'current',
    ],
  },
]);
```

## Common errors

| Error | Cause | Fix |
|-------|--------|-----|
| **`chain: Method "<name>" does not exist`** | Typo / private API | Use public **`ApexPainter`** methods only. |
| **`chain: Operation … did not return a buffer`** | Method returned unsupported shape | Pick methods that return **`Buffer`** or **`{ buffer }`**. |
| **`chain: No buffer was produced`** | Empty ops / nothing returned | Ensure first step yields a raster buffer. |
