# Your First Image

## What this does

You will render **text on a solid background** and save a **PNG** file — the smallest complete pipeline in Apexify.js.

## When to use it

Perfect when you are **new to the library**, sanity-checking an install, or shipping a **simple title card** before layering shapes or charts.

## Quick example

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from "apexify.js";
import fs from "fs";

const painter = new ApexPainter();

const { buffer } = await painter.createCanvas({
  width: 800,
  height: 450,
  colorBg: "#0f172a",
});

const withText = await painter.createText(
  {
    text: "Hello Apexify.js",
    x: 400,
    y: 225,
    font: { size: 48, family: "Arial" },
    color: "#ffffff",
    textAlign: "center",
    textBaseline: "middle",
  },
  buffer
);

fs.writeFileSync("output.png", withText);
```
</ts>
<js>
```javascript
const { ApexPainter } = require("apexify.js");
const fs = require("fs");

(async () => {
  const painter = new ApexPainter();
  const { buffer } = await painter.createCanvas({
    width: 800,
    height: 450,
    colorBg: "#0f172a",
  });
  const withText = await painter.createText(
    {
      text: "Hello Apexify.js",
      x: 400,
      y: 225,
      font: { size: 48, family: "Arial" },
      color: "#ffffff",
      textAlign: "center",
      textBaseline: "middle",
    },
    buffer
  );
  fs.writeFileSync("output.png", withText);
})();
```

## How it works

1. **`new ApexPainter()`** — creates your painter instance.
2. **`createCanvas`** — allocates an **800×450** image filled with **`#0f172a`**.
3. **`createText`** — draws **“Hello Apexify.js”** centered around **(400, 225)** using **`textAlign: "center"`** and **`textBaseline: "middle"`**.
4. **`fs.writeFileSync`** — writes the returned buffer to **`output.png`**.

## Common changes

- **Bigger title:** increase **`font.size`** (try **`64`**).
- **Different placement:** move **`x`** / **`y`** — center stays at half of **`width`** / **`height`** when aligned center/middle.
- **Second line:** pass an **array** of text objects to **`createText`** (see **[Add Text](/docs#add-text)**).

## Common mistakes

- **Skipping `await`** — always **`await`** **`createCanvas`** and **`createText`**.
- **Saving the wrong variable** — save **`withText`**, not the original canvas buffer, after you draw text.
- **Font not found** — stick to common families like **Arial** first; custom fonts come later.
