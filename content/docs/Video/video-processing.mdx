# Comprehensive Video Processing

Apexify.js provides **35+ comprehensive video processing features** through the `createVideo()` method. This unified API gives you complete control over video manipulation, frame extraction, effects, transitions, and professional editing capabilities.

<Alert type="info" title="FFmpeg Required">
All video processing features require FFmpeg to be installed on your system. See the [FFmpeg Installation Guide](#ffmpeg-installation) for setup instructions.
</Alert>

## Overview

The `createVideo()` method is a comprehensive video processing API that supports:

- **Frame Extraction**: Extract single frames, multiple frames, or all frames, and create videos from frames
- **Video Information**: Get detailed video metadata and format detection
- **Format Conversion**: Convert between video formats with platform-optimized presets
- **Video Editing**: Trim, crop, rotate, merge, and replace video segments
- **Effects & Filters**: Apply visual effects, color correction, and LUT color grading
- **Audio Processing**: Extract audio, adjust volume, mute (full or partial), normalize audio levels
- **Transitions**: Professional transitions between videos (fade, wipe, slide, zoom, and more)
- **Text Animations**: Add animated text overlays with custom fonts and animations
- **Advanced Features**: Stabilization, scene detection, time-lapse, freeze frames, and more

    width: 1920,         // Canvas width (default: video width)
    height: 1080,        // Canvas height (default: video height)
    outputFormat: 'png', // 'jpg' | 'png'
    quality: 2           // JPEG quality 1-31 (lower = better)

import fs from 'fs';

    source: './video.mp4',
    getInfo: true
  });
  console.log('Video info:', info);

  // 2. Extract frame for thumbnail
  const frame = await painter.createVideo({
    source: './video.mp4',
    extractFrame: {
      time: info.duration * 0.5,  // Middle of video
      outputFormat: 'png'
    }
  });
  fs.writeFileSync('thumbnail.png', frame.buffer);

  // 3. Compress video
  const compressed = await painter.createVideo({

    compress: {
      outputPath: './compressed.mp4',
      quality: 'medium'
    }
  });

  // 4. Add watermark
  const watermarked = await painter.createVideo({
    source: './compressed.outputPath',
    addWatermark: {
      watermarkPath: './logo.png',
      position: 'bottom-right',
      opacity: 0.7,
      outputPath: './final.mp4'
    }
  });

}
