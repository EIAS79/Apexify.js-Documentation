# Paths, pixels & hit testing

Low-level tools on top of **`@napi-rs/canvas`**: build **`Path2D`** objects from a command list, **stroke/fill** them onto an existing PNG scene, read/write **RGBA** pixels, and answer **point-in-shape** questions for interactive UIs.

**Implementation:** `lib/Canvas/extended/Path2DCreator.ts`, **`PixelDataCreator.ts`**, **`HitDetectionCreator.ts`** · command DSL **`lib/Canvas/utils/core/pathCmd.ts`**.

---

## Quick picks

**Stroke + fill a rounded rectangle on a canvas**

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const { buffer } = await painter.createCanvas({
  width: 480,
  height: 320,
  colorBg: '#0f172a',
});

const outlined = await painter.drawPath(
  buffer,
  [
    { type: 'roundedRect', x: 80, y: 70, width: 320, height: 180, radius: 24 },
  ],
  {
    stroke: { color: '#38bdf8', width: 4, lineJoin: 'round' },
    fill: { color: 'rgba(56, 189, 248, 0.12)', rule: 'nonzero' },
  }
);

fs.writeFileSync('frame.png', outlined);
```
</ts>
<js>
```javascript
const { ApexPainter } = require('apexify.js');
const fs = require('fs');

(async () => {
  const painter = new ApexPainter();
  const { buffer } = await painter.createCanvas({
    width: 480,
    height: 320,
    colorBg: '#0f172a',
  });

  const outlined = await painter.drawPath(
    buffer,
    [{ type: 'roundedRect', x: 80, y: 70, width: 320, height: 180, radius: 24 }],
    {
      stroke: { color: '#38bdf8', width: 4, lineJoin: 'round' },
      fill: { color: 'rgba(56, 189, 248, 0.12)', rule: 'nonzero' },
    }
  );

  fs.writeFileSync('frame.png', outlined);
})();
```
</js>
</CodeSwitcher>
