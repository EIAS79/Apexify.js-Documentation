# Shapes

Apexify.js supports 8 built-in shape types that you can use instead of loading image files. Shapes are created using the `createImage()` method with a shape name as the `source` property.

## Available Shapes

- `'rectangle'` - Standard rectangle
- `'square'` - Perfect square
- `'circle'` - Perfect circle
- `'triangle'` - Equilateral triangle
- `'trapezium'` - Trapezoid shape
- `'star'` - 5-pointed star
- `'heart'` - Heart shape with bezier curves
- `'polygon'` - Custom polygon with configurable sides

## Basic Shape Usage

All shapes use the same basic structure with the `shape` property to define fill and styling:

<CodeSwitcher>
<ts>
```typescript
const shape = await painter.createImage({
  source: 'circle',  // Shape type
  x: 100,
  y: 100,
  width: 200,
  height: 200,
  shape: {
    fill: true,      // Fill the shape
    color: '#ff6b6b' // Fill color
  }
}, canvasBuffer);
```
</ts>


## Gradient Fills

All shapes support gradient fills using the `gradient` property:

<CodeSwitcher>
<ts>
```typescript
// Linear gradient
const gradientShape = await painter.createImage({
  source: 'circle',
  x: 100,
  y: 100,
  width: 200,
  height: 200,
  shape: {
    fill: true,
    gradient: {
      type: 'linear',
      colors: [
        { stop: 0, color: '#ff6b6b' },
        { stop: 1, color: '#4ecdc4' }
      ],
      startX: 0,
      startY: 0,
      endX: 200,
      endY: 200
    }
  }
}, canvasBuffer);

// Radial gradient
const radialGradientShape = await painter.createImage({
  source: 'star',
  x: 100,
  y: 100,

      startY: 100,
      startRadius: 0,
      endX: 100,
```

## Combining Shapes with Effects

Shapes work seamlessly with all image effects like shadows, strokes, rotation, and more:

<CodeSwitcher>
<ts>
```typescript
const styledShape = await painter.createImage({
  source: 'heart',
  x: 200,
  y: 200,
  width: 200,
  height: 200,
  shape: {
    fill: true,
    gradient: {
      type: 'radial',
      colors: [
        { stop: 0, color: '#ff6b6b' },
        { stop: 1, color: '#ff1744' }
      ]
    }
  },
  shadow: {
    color: 'rgba(0,0,0,0.5)',
    offsetX: 10,
    offsetY: 10,
    blur: 20
  },
  stroke: {
    color: '#ffffff',
    width: 5
  },
  rotation: 15,
  borderRadius: 10
}, canvasBuffer);
```
