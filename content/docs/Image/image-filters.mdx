# Image Filters

Apexify.js provides 22+ professional image filters that you can apply to images and shapes. Filters are applied using the `filters` array property in `ImageProperties`.

## Overview

Filters are applied in the order they appear in the array. You can combine multiple filters for complex effects:

<CodeSwitcher>
<ts>
```typescript
const filteredImage = await painter.createImage({
  source: './image.jpg',
  x: 100,
  y: 100,
  width: 400,
  height: 300,
  filters: [
    { type: 'grayscale' },
    { type: 'contrast', value: 1.2 },
    { type: 'brightness', value: 0.1 }
  ]
}, canvasBuffer);
```
</ts>
<js>
```javascript
const filteredImage = await painter.createImage({
  source: './image.jpg',
  x: 100,

  width: 400,
  height: 300,
```

## Global Filter Intensity

Use `filterIntensity` to control all filters at once:

<CodeSwitcher>
<ts>
```typescript

    { type: 'gaussianBlur', intensity: 5 },
    { type: 'contrast', value: 1.5 }
```

## Filter Order

Control when filters are applied using `filterOrder`:

- `'pre'` - Apply filters before transformations (rotation, scaling)
- `'post'` - Apply filters after transformations (default)

<CodeSwitcher>
<ts>
```typescript
// Apply filters before rotation
const preFilter = await painter.createImage({
  source: './image.jpg',
  x: 100,
  y: 100,
  width: 400,
  height: 300,
  rotation: 45,
  filters: [
    { type: 'gaussianBlur', intensity: 5 }
  ],
  filterOrder: 'pre'
}, canvasBuffer);

// Apply filters after rotation (default)
const postFilter = await painter.createImage({
  source: './image.jpg',
  x: 100,
  y: 100,
  width: 400,
  height: 300,
  rotation: 45,
  filters: [
    { type: 'gaussianBlur', intensity: 5 }
  ],
  filterOrder: 'post'  // or omit for default
}, canvasBuffer);
```
</ts>
<js>
```javascript
// Apply filters before rotation
const preFilter = await painter.createImage({
  source: './image.jpg',
  x: 100,
  y: 100,
  width: 400,
  height: 300,
  rotation: 45,
  filters: [
    { type: 'gaussianBlur', intensity: 5 }
  ],
  filterOrder: 'pre'
}, canvasBuffer);

// Apply filters after rotation (default)
const postFilter = await painter.createImage({
  source: './image.jpg',
  x: 100,
  y: 100,
  width: 400,
  height: 300,
  rotation: 45,
  filters: [
    { type: 'gaussianBlur', intensity: 5 }
  ],
  filterOrder: 'post'  // or omit for default
}, canvasBuffer);
```
</js>

## Complete Filter Reference

| Filter | Properties | Description |
|--------|-----------|-------------|
| `gaussianBlur` | `intensity` | Smooth, uniform blur |
| `motionBlur` | `intensity`, `angle` | Directional motion blur |
| `radialBlur` | `intensity`, `centerX`, `centerY` | Circular blur from center |
| `sharpen` | `intensity` | Enhance details and edges |
| `brightness` | `value` | Adjust brightness (-1 to 1) |
| `contrast` | `value` | Adjust contrast (0 to 2+) |
| `saturation` | `value` | Adjust color saturation (0 to 2+) |
| `hueShift` | `value` | Shift hue in degrees (0-360) |
| `grayscale` | - | Convert to grayscale |
| `sepia` | - | Apply sepia tone |
| `invert` | - | Invert all colors |
| `posterize` | `levels` | Reduce color levels (2-256) |
| `pixelate` | `size` | Pixelate effect (pixel size) |
| `noise` | `intensity` | Add random noise (0-1) |
| `grain` | `intensity` | Add film grain (0-1) |
| `edgeDetection` | `intensity` | Detect and highlight edges |
| `emboss` | `intensity` | Create 3D embossed effect |
