# Text Decorations

Apexify.js provides comprehensive text decoration options including bold, italic, underline, overline, strikethrough, and highlight effects. Decorations can use solid colors or gradients for enhanced visual appeal.

## Overview

Text decorations enhance your text with visual styling:
- **Bold** - Make text bold
- **Italic** - Make text italic
- **Underline** - Add underline with color or gradient
- **Overline** - Add overline with color or gradient
- **Strikethrough** - Add strikethrough with color or gradient
- **Highlight** - Add background highlight with color or gradient

## Underline

Add underline decoration with simple boolean or advanced configuration:

<CodeSwitcher>
<ts>
```typescript
// Simple underline (uses text color)
const simpleUnderline = await painter.createText({
  text: 'Underlined Text',
  x: 100,
  y: 100,
  fontSize: 24,
  underline: true,
  color: '#ffffff'
}, canvasBuffer);

// Custom underline color
const coloredUnderline = await painter.createText({
  text: 'Colored Underline',
  x: 100,
  y: 150,
  fontSize: 24,
  underline: {
    color: '#ff6b6b',
    width: 2
  },
  color: '#ffffff'
}, canvasBuffer);

// Gradient underline
const gradientUnderline = await painter.createText({
  text: 'Gradient Underline',
  x: 100,
  y: 200,
  fontSize: 24,
  underline: {
    gradient: {
      type: 'linear',
      colors: [
        { stop: 0, color: '#ff6b6b' },
        { stop: 1, color: '#4ecdc4' }
      ],
      startX: 0,
      startY: 0,
      endX: 200,
      endY: 0
    },
    width: 3
  },
  color: '#ffffff'
}, canvasBuffer);
```

## Complete Example

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const canvas = await painter.createCanvas({
  width: 1200,
  height: 800,
  colorBg: '#1a1a1a'
});

const result = await painter.createText([
  {
    text: 'Bold & Underlined',
    x: 100,
    y: 100,
    fontSize: 28,
    bold: true,
    underline: { color: '#ff6b6b', width: 2 },
    color: '#ffffff'
  },
  {
    text: 'Italic & Highlighted',
    x: 100,
    y: 200,
    fontSize: 28,
    italic: true,
    highlight: { color: '#ffd700', opacity: 0.4 },
    color: '#000000'
  },
  {
    text: 'Gradient Decorations',
    x: 100,
    y: 300,
    fontSize: 28,
    underline: {
      gradient: {
        type: 'linear',
        colors: [
          { stop: 0, color: '#ff6b6b' },
          { stop: 1, color: '#4ecdc4' }
        ]
      },
      width: 3
    },
    highlight: {
      gradient: {
        type: 'radial',
        colors: [
          { stop: 0, color: '#ffd700' },
          { stop: 1, color: '#ff6b6b' }
        ]
      },
      opacity: 0.3
    },
    color: '#ffffff'
  }
], canvas.buffer);

fs.writeFileSync('decorations.png', result);
```
</ts>
<js>
```javascript
const { ApexPainter } = require('apexify.js');
const fs = require('fs');

const painter = new ApexPainter();

async function createDecorations() {
  const canvas = await painter.createCanvas({
    width: 1200,
    height: 800,
    colorBg: '#1a1a1a'
  });

  const result = await painter.createText([
    {
      text: 'Bold & Underlined',
      x: 100,
      y: 100,
      fontSize: 28,
      bold: true,
      underline: { color: '#ff6b6b', width: 2 },
      color: '#ffffff'
    },
    {
      text: 'Italic & Highlighted',
      x: 100,
      y: 200,
      fontSize: 28,
      italic: true,
      highlight: { color: '#ffd700', opacity: 0.4 },
      color: '#000000'
    },
    {
      text: 'Gradient Decorations',
      x: 100,
      y: 300,
      fontSize: 28,
      underline: {
        gradient: {
          type: 'linear',
          colors: [
            { stop: 0, color: '#ff6b6b' },
            { stop: 1, color: '#4ecdc4' }
          ]
        },
        width: 3
      },
      highlight: {
        gradient: {
          type: 'radial',
          colors: [
            { stop: 0, color: '#ffd700' },
            { stop: 1, color: '#ff6b6b' }
          ]
        },
        opacity: 0.3
      },
      color: '#ffffff'
    }
  ], canvas.buffer);

  fs.writeFileSync('decorations.png', result);
}

createDecorations();
```
</js>
