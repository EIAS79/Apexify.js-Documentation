# Text Spacing and Alignment

Apexify.js provides comprehensive control over text spacing, alignment, and wrapping. Fine-tune letter spacing, word spacing, line height, and control how text aligns both horizontally and vertically.

## Overview

Spacing and alignment features include:
- **Line Height** - Control vertical spacing between lines
- **Letter Spacing** - Space between individual characters
- **Word Spacing** - Space between words
- **Text Wrapping** - Automatic text wrapping with `maxWidth`
- **Text Alignment** - Horizontal alignment (left, center, right)
- **Text Baseline** - Vertical baseline positioning

## Line Height

Control the vertical spacing between lines of text:

<CodeSwitcher>
<ts>
```typescript
// Tight line spacing
const tight = await painter.createText({
  text: 'Line 1\nLine 2\nLine 3',
  x: 100,
  y: 100,
  fontSize: 24,
  lineHeight: 1.2,  // Tighter spacing
  color: '#ffffff'
}, canvasBuffer);

// Normal line spacing (default: 1.4)
const normal = await painter.createText({
  text: 'Line 1\nLine 2\nLine 3',
  x: 100,
  y: 200,

  lineHeight: 1.4,  // Default

// Loose line spacing

  text: 'Line 1\nLine 2\nLine 3',
  x: 100,
  y: 300,
  fontSize: 24,
  lineHeight: 2.0,  // Loose spacing
  color: '#ffffff'
}, canvasBuffer);
```

## Text Wrapping

Automatically wrap text to fit within a maximum width:

<CodeSwitcher>
<ts>
```typescript
// Text with wrapping
const wrapped = await painter.createText({
  text: 'This is a long text that will automatically wrap to fit within the specified maximum width. The text will break at word boundaries to ensure readability.',
  x: 100,
  y: 100,
  fontSize: 20,
  maxWidth: 400,  // Maximum width in pixels
  color: '#ffffff',
  lineHeight: 1.5
}, canvasBuffer);

// Text without wrapping (default)
const noWrap = await painter.createText({
  text: 'This text will not wrap and will continue on a single line',
  x: 100,
  y: 300,
  fontSize: 20,
  // No maxWidth = no wrapping
  color: '#ffffff'
}, canvasBuffer);
```

<Alert type="tip" title="Perfect Centering">
For perfect text centering, use `textAlign: 'center'` with `textBaseline: 'middle'`. The x and y coordinates will be the exact center point.
</Alert>

## Complete Example

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const canvas = await painter.createCanvas({
  width: 1200,
  height: 800,
  colorBg: '#1a1a1a'
});

const result = await painter.createText([
  // Centered title
  {
    text: 'Perfectly Centered Title',
    x: 600,
    y: 100,
    fontSize: 48,
    bold: true,
    textAlign: 'center',
    textBaseline: 'middle',
    color: '#ffffff'
  },
  // Wrapped paragraph
  {
    text: 'This is a long paragraph that demonstrates automatic text wrapping. The text will automatically break at word boundaries to fit within the specified maximum width, ensuring readability and proper formatting.',
    x: 100,
    y: 250,
    fontSize: 18,
    maxWidth: 500,
    lineHeight: 1.6,
    letterSpacing: 0.3,
    color: '#cccccc'
  },
  // Right-aligned text
  {
    text: 'Right Aligned Text',
    x: 1100,
    y: 500,
    fontSize: 24,
    textAlign: 'right',
    textBaseline: 'top',
    color: '#ffffff'
  },
  // Wide letter spacing
  {
    text: 'W I D E   S P A C I N G',
    x: 600,
    y: 600,
    fontSize: 32,
    textAlign: 'center',
    textBaseline: 'middle',
    letterSpacing: 8,
    color: '#ff6b6b'
  }
], canvas.buffer);

fs.writeFileSync('spacing-alignment.png', result);
```
</ts>

```javascript
      x: 100,
      y: 250,
      fontSize: 18,
      maxWidth: 500,
      lineHeight: 1.6,
      letterSpacing: 0.3,
      color: '#cccccc'
    },
    // Right-aligned text
    {
      text: 'Right Aligned Text',
      x: 1100,
      y: 500,
      fontSize: 24,

      color: '#ff6b6b'

createSpacingExample();
```
