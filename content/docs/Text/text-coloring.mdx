# Text Coloring

Apexify.js provides powerful text coloring options including solid colors, gradients, and opacity control. Create vibrant, eye-catching text with gradient fills and transparency effects.

## Overview

Text coloring features include:
- **Solid Colors** - Hex, RGB, RGBA, HSL, and named colors
- **Gradient Fills** - Linear and radial gradients
- **Opacity** - Control text transparency (0-1)

## Gradient Fills

Create stunning gradient text effects with linear or radial gradients:

### Linear Gradients

<CodeSwitcher>
<ts>
```typescript
// Horizontal linear gradient
const horizontal = await painter.createText({
  text: 'Horizontal Gradient',
  x: 100,
  y: 100,
  fontSize: 48,
  gradient: {
    type: 'linear',
    colors: [
      { stop: 0, color: '#ff6b6b' },
      { stop: 1, color: '#4ecdc4' }

    startX: 0,
    startY: 0,
```

## Complete Example

<CodeSwitcher>
<ts>
```typescript
import { ApexPainter } from 'apexify.js';
import fs from 'fs';

const painter = new ApexPainter();

const canvas = await painter.createCanvas({
  width: 1200,
  height: 800,
  colorBg: '#1a1a1a'
});

const result = await painter.createText([
  {
    text: 'Solid Color',
    x: 100,
    y: 100,
    fontSize: 32,
    color: '#ff6b6b'
  },
  {
    text: 'Linear Gradient',
    x: 100,
    y: 200,
    fontSize: 48,
    gradient: {
      type: 'linear',
      colors: [
        { stop: 0, color: '#ff6b6b' },
        { stop: 1, color: '#4ecdc4' }
      ],
      startX: 0,
      startY: 0,
      endX: 400,
      endY: 0
    }
  },
  {
    text: 'Radial Gradient',
    x: 100,
    y: 300,
    fontSize: 48,
    gradient: {
      type: 'radial',
      colors: [
        { stop: 0, color: '#ffd700' },
        { stop: 1, color: '#ff1744' }
      ],
      startX: 200,
      startY: 25,
      startRadius: 0,
      endX: 200,
      endY: 25,
      endRadius: 100
    }
  },
  {
    text: 'Transparent Text',
    x: 100,
    y: 400,
    fontSize: 32,
    color: '#ffffff',
    opacity: 0.5
  }
], canvas.buffer);

fs.writeFileSync('text-coloring.png', result);
```

```javascript
const { ApexPainter } = require('apexify.js');

  const canvas = await painter.createCanvas({

  const result = await painter.createText([
    {
      text: 'Solid Color',
      x: 100,
      y: 100,
      fontSize: 32,
      color: '#ff6b6b'
    },
    {
      text: 'Linear Gradient',
      x: 100,
      y: 200,
      fontSize: 48,
      gradient: {
        type: 'linear',
        colors: [
          { stop: 0, color: '#ff6b6b' },
          { stop: 1, color: '#4ecdc4' }
        ],
        startX: 0,
        startY: 0,
        endX: 400,
        endY: 0
      }
    },
    {
      text: 'Radial Gradient',
      x: 100,
      y: 300,
      fontSize: 48,
      gradient: {
        type: 'radial',
        colors: [
          { stop: 0, color: '#ffd700' },
          { stop: 1, color: '#ff1744' }
        ],
        startX: 200,
        startY: 25,
        startRadius: 0,
        endX: 200,
        endY: 25,
        endRadius: 100
      }
    },
    {
      text: 'Transparent Text',
      x: 100,
      y: 400,
      fontSize: 32,
      color: '#ffffff',
      opacity: 0.5
    }
  ], canvas.buffer);

  fs.writeFileSync('text-coloring.png', result);
}

createColoringExample();
```
