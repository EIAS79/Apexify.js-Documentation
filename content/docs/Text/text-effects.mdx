# Text Effects

Apexify.js provides powerful text effects including glow, shadow, and stroke. These effects add depth, dimension, and visual appeal to your text, and all support gradient fills for stunning results.

## Overview

Text effects include:
- **Glow** - Add a glowing effect around text
- **Shadow** - Add drop shadows for depth
- **Stroke** - Add outlines/borders around text

All effects support both solid colors and gradients!
