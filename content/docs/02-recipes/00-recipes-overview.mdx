## Folder conventions



| Rule | Why |

|------|-----|

| **One recipe per file** | Stable anchors; easier reviews |

| **Name** | **`recipe-<topic>.mdx`** (kebab-case) |

| **Structure** | Title → **Goal** → **Prerequisites** → **Steps** → **Full example** → **See also** (guide links) |

| **Anchor** | Heading **`{#recipe-<topic>}`** on the **`#`** title for deep links |

## Recipe backlog (add as `recipe-*.mdx`)



When you publish a file, add a row with its anchor link.



| Recipe file (planned) | Builds | Main APIs |

|----------------------|--------|-----------|

| **`recipe-social-banner.mdx`** | Twitter/X / LinkedIn covers | **`createCanvas`**, **`createImage`**, **`createText`**, **`save`** |

| **`recipe-quote-card.mdx`** | Shareable quote PNGs | Gradients, **`createText`**, shadows |

| **`recipe-discord-welcome.mdx`** | Welcome cards | Layers, **`createImage`**, styled text |

| **`recipe-thumbnail-sheet.mdx`** | Grid of thumbs | **`createCanvas`** layout loop **or** **`createCollage`** |

| **`recipe-chart-daily.mdx`** | Automated KPI image | **`createChart`**, **`save`** |

| **`recipe-gif-loader.mdx`** | Small animated GIF | **`createGIF`** **or** **`animate`** |

| **`recipe-batch-assets.mdx`** | Many sizes from one template | **`batch`**, **`chain`**, **`saveMultiple`** |

| **`recipe-video-poster.mdx`** | Poster frame MP4 | **`createVideo`** **`extractFrame`** **or** **`extractFrameAtTime`** |



Starter ideas only — extend the table as you ship files.
