# Chart Image

**Goal:** Export a chart as **PNG** via **`createChart`** (or comparison/combo APIs).

## Flow

1. Read **[Charts hub](/docs#charts)** and **[Create chart](/docs#charts-create-chart)**.  
2. Pick **`bar`**, **`line`**, **`pie`**, **`horizontalBar`**, etc.  
3. Tune **`appearance`** (backgrounds, legends, axes).  
4. Write buffer with **`save`** / **`fs`**.

Advanced compositions: **[Charts comparison & combo](/docs#charts-comparison-combo)**.
