# Quote Image

**Goal:** Square or portrait graphic with a quote, attribution, and subtle framing.

## Flow

1. **`createCanvas`** — neutral or paper-toned **`colorBg`** / **`gradientBg`**.  
2. **`createText`** — quote body + attribution; use **`textAlign`** / **`textBaseline`**.  
3. Optional shapes via **`createImage`** for rules or quotation marks.  

Deep styling: **[Decorations & effects](/docs#text-decorations-effects)**.
