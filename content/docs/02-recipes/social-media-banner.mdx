# Social Media Banner

**Goal:** A wide canvas with gradient background and headline text — typical **1200×630** or similar.

## Flow

1. **[Create canvas](/docs#create-canvas)** — set dimensions; use **`gradientBg`** or stacks from **[Backgrounds](/docs#canvas-bg-layers)**.  
2. **[Create text](/docs#create-text)** — title + subtitle with alignment.  
3. Optional **[Create image](/docs#create-image-and-shapes)** — logo chip or accent shape.  
4. **[Save & output](/docs#save-and-output)**.
