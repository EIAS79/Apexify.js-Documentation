# Product Card

**Goal:** Product tile — background panel, title, price row, optional badge.

## Flow

1. **`createCanvas`** — card aspect ratio.  
2. **`createImage`** — rounded panel (**`stroke`**, **`shadow`**, **`shape`**) per **[Image shapes](/docs#image-shapes)**.  
3. **`createText`** — title + meta.  
4. **`save`** — PNG for web or catalog export.

Filters (optional): **[Image filters](/docs#image-filters-groups-and-helpers)**.
