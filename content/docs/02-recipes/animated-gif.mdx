# Animated GIF

**Goal:** Encode multiple frames into a GIF.

## Flow

1. Build each frame with **`createCanvas`** / **`createImage`** / **`createText`**.  
2. Use **`createGIF`** — see **[Create GIF](/docs#gif-create)** and **[GIF & animation](/docs#gif-and-animation)**.  
3. For timeline helpers, **[Animate & Frame](/docs#gif-animate)**.
