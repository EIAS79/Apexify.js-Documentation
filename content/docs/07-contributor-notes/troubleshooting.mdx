## Common pitfalls

1. **Relative paths** — Most loaders join **`process.cwd()`**. Bundle/working-directory surprises ≠ library bugs.
2. **`blend` argument order** — Layers stack **over** explicit base buffer ([Standalone ops](/docs#standalone-image-operations)).
3. **GIF return unions** — Always narrow **`createGIF`** results ([GIF](/docs#gif-and-animation)).
4. **Video URLs vs `#` helpers** — Prefer **`Buffer`** or absolute paths when a branch does not call **`resolveVideoSource`** ([Video hub](/docs#video-ffmpeg), [FFmpeg internals](/docs#video-ffmpeg-internals)).
5. **Offline CI** — npm entry **`fetch`** version check may fail silently; Imgur **`url()`** requires network + trusting bundled credentials.
