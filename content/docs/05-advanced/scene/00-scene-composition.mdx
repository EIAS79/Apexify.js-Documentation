# Scene — overview {#00-scene-composition}

Back: [Advanced hub](/docs#00-advanced-overview) · [Feature tracks](/docs#01-feature-tracks)

This page is the **entry point** for Scene: a **declarative layer graph** where you set **width**, **height**, an optional **background**, and an ordered **`layers`** array. **`SceneCreator`** composites everything into **one PNG `Buffer`** in a single pass. From there you can call **`renderScene`**, **`renderSceneToGIF`**, or **`renderSceneToVideoFrames`** depending on whether you need a still, an animated GIF, or a video file.

**Implementation:** `lib-next/scene/scene-creator.ts` (**`SceneCreator`**, **`SceneBuilder`**), `gif-scene.ts`, `video-scene.ts`, `render-scene-to-video.ts` · **Types:** `lib-next/types/scene.ts`

**On `ApexPainter`:** `createScene`, `renderScene`, `renderSceneToGIF`, `renderSceneToVideoFrames`

---

## What you get in one sentence each

| Output | API | Idea |
|--------|-----|------|
| **PNG** | **`renderScene`** or **`SceneBuilder.render()`** | Final poster / card / dashboard bitmap. |
| **GIF** | **`renderSceneToGIF`** | Renders the scene once, optionally **prepends** that frame, merges **`gifFrames`** (with **`repeat`**), runs **`createGIF`**. |
| **Video** | **`renderSceneToVideoFrames`** | Same composed PNG, merged with **`createFromFrames`** + optional **`framesWithRepeats`** → FFmpeg. |

---

## When to use Scene

| Situation | Recommendation |
|-----------|----------------|
| One background + a couple of images/text | **`createCanvas`** + **`createImage`** / **`createText`** — fewer moving parts. |
| Many layers, **charts as layers**, **nested cards** (`surface`), one export | **Scene** |
| Same visual design → **GIF** or **MP4** | **`renderSceneToGIF`** / **`renderSceneToVideoFrames`** after designing **`layers`**. |

Scene reuses the same engines as the rest of the library (**`@napi-rs/canvas`**, **`ChartCreator`**, **`ImageCreator`**, **`TextCreator`**, **`Path2DCreator`**, **`CanvasCreator`** for backgrounds).

## Minimal example

```typescript
import { ApexPainter } from "apexify.js";
import fs from "fs";

const painter = new ApexPainter();

const png = await painter
  .createScene(640, 360)
  .setBackground({ colorBg: "#0f172a" })
  .addLayer({
    type: "text",
    texts: [{ content: "Hello, Scene", x: 48, y: 160, fontSize: 36, color: "#e2e8f0" }],
  })
  .render();

fs.writeFileSync("./scene.png", png);
```
