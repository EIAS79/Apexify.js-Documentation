# Scene composition — hub {#00-scene-hub}

Back: [Advanced hub](/docs#00-advanced-overview) · [Feature tracks](/docs#01-feature-tracks)

The **Scene** API is a **declarative layer graph**: **width**, **height**, optional **background**, and an ordered **`layers`** list. **`SceneCreator`** paints everything into **one PNG `Buffer`** in one pass. You can pipe that raster into **GIF** (`renderSceneToGIF`) or **video** (`renderSceneToVideoFrames`).

**Code:** `lib-next/scene/scene-creator.ts`, `gif-scene.ts`, `video-scene.ts`, `render-scene-to-video.ts` · **Types:** `lib-next/types/scene.ts`  
**`ApexPainter`:** `createScene`, `renderScene`, `renderSceneToGIF`, `renderSceneToVideoFrames`

---

## When to use Scene

| Goal | Prefer |
|------|--------|
| Simple poster: one bg + a few draws | **`createCanvas`** + **`createImage`** / **`createText`** |
| Many layers, **charts**, **nested cards**, one export | **Scene** |
| Same layout → **GIF** or **MP4** | **`renderSceneToGIF`** / **`renderSceneToVideoFrames`** |

## Minimal end-to-end

```typescript
import { ApexPainter } from "apexify.js";
import fs from "fs";

const painter = new ApexPainter();

const png = await painter
  .createScene(640, 360)
  .setBackground({ colorBg: "#0f172a" })
  .addLayer({
    type: "text",
    texts: [{ content: "Hello, Scene", x: 48, y: 160, fontSize: 36, color: "#e2e8f0" }],
  })
  .render();

fs.writeFileSync("./scene.png", png);
```
